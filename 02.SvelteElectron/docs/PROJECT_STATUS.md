## 5) 리스크/주의사항
- **Rust 빌드 필수**: Rust 로직 수정 시 `npm run build:native` 후 Electron 재시작 필요
- **napi-rs 자동생성 파일**: `index.d.ts`, `index.js`는 `build:native` 시 자동 재생성 — 직접 편집 금지
- **IPC await 필수**: `window.projectB!.*()` 호출은 반드시 `await` 필요
- 한글 인코딩 깨짐 이력:
  - Svelte 파일/템플릿 편집 시 UTF-8 유지 필수
- 매니페스트/인덱스 동기화:
  - `resource/data/master/_manifest.json`
  - `resource/data/master/entities/players/_index.json`
  - 데이터 갱신 스크립트 실행 시 자동 변경될 수 있음
