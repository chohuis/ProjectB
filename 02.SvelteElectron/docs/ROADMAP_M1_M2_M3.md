# ROADMAP (M1 / M2 / M3)

## M1: 고교-대학 전환 흐름
목표: 고교 시즌 종료 시점에서 진로 분기가 끊기지 않고 일관되게 이어지도록 완성

### 범위
- W50 진로 선택
- W51 드래프트
- W52 폴백 지원/결과/최종 선택

### Done 기준
- 고교 3학년 플레이에서 W50~W52 이벤트가 100% 순서대로 발생
- 미지명 시 대학/독립/체육부대 지원 결과가 표시되고 선택 가능
- 선택 결과가 세이브/다음 시즌 상태에 반영

### 남은 작업
- 합격/불합격 판정식 데이터 외부화
- 결과 메시지/뉴스 템플릿 확장
- 회귀 시나리오 문서화

---

## M2: 시즌 종료 안정화 + NPC 라이프사이클
목표: 시즌 종료 후 학년/로스터/은퇴 정리가 안정적으로 적용되도록 고정

### 범위
- 시즌 종료 모달에서 동기화 실행
- NPC 오프시즌 정규화(은퇴/로스터 캡)
- 결과 로그/뉴스 기록

### Done 기준
- 시즌 종료 이후 NPC 상태가 규칙에 맞게 전환
- 로스터 과밀/미달이 로그로 확인 가능
- 다음 시즌 진입 시 상태 불일치 없음

### 남은 작업
- 로스터 규칙 JSON 외부화
- 라이프사이클 리포트 UI 개선
- 다년 시뮬레이션 점검

---

## M3: 프로 리그 디테일 고도화
목표: 프로 커리어 장기 플레이가 자연스럽도록 규칙/콘텐츠를 확장

### 예정 범위
- 2군 체계(경량/실시뮬 중 택1)
- 강등 보호 규칙(부상복귀/유망주)
- 은퇴/방출/이적 정교화
- 뉴스/연감 기록 시스템 강화

### 착수 전 선행
- M1/M2 통합 점검 완료
- 리그별 로스터 정원/포지션 최소 인원 기획 확정
- 군복무 NPC 적용 범위(주인공만 vs 전원) 정책 확정
