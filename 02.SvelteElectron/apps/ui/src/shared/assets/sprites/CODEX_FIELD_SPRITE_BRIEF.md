# 야구 게임 필드 탑뷰 스프라이트 제작 의뢰서

---

## 핵심 요약

| 항목 | 내용 |
|---|---|
| 형식 | PNG, 투명 배경 |
| 원본 크기 | **16×18px** (1도트 = 1px) |
| 화면 렌더 | 약 2.5배 확대 표시 (`image-rendering: pixelated`) |
| 스타일 | GBC(게임보이 컬러) 픽셀아트 — 포켓몬 금·은 트레이너 수준 |
| 수량 | **4개** |

---

## 1. 시점 — 탑뷰 (필드 전체 조감)

야구 경기장을 **위에서 약 45~60도 내려다보는 시점**입니다.

```
         [센터]
  [좌익수]      [우익수]
       [유격수][2루수]
    [3루수]          [1루수]
           [투수]
           [포수]
           [타자]
```

캐릭터는 이 시점에서 보이는 작은 인물입니다.  
**정면/측면이 아닌 탑뷰** — 머리(모자)가 가장 크게 보이고, 얼굴과 몸통이 아래로 보입니다.

---

## 2. 스타일 기준

포켓몬 금·은(GBC) 또는 포켓몬 루비·사파이어(GBA)의 **필드 위 트레이너 스프라이트** 수준.

- 명확한 다크 아웃라인 (1px)
- 모자·얼굴·유니폼·바지 각 영역 하이라이트·그림자 1단계 이상
- 안티앨리어싱 없음 (픽셀 경계 선명)
- 치비 비율 — 머리(모자)가 전체의 40~50% 차지

---

## 3. 요청 파일 (3개)

> 네 파일 모두 **기본 실루엣(체형·머리·다리)은 동일**합니다.  
> 손에 들고 있는 것(글러브 / 방망이 / 없음)과 방망이 방향만 다릅니다.

## 4. 픽셀 구성 참고 (16×18, 공통)

```
상단 6행:  모자/헬멧 (캡 + 챙) — 가장 넓은 부분
중간 4행:  얼굴 (작은 눈 2개, 피부색)
하단 8행:  유니폼 상의 + 바지 + 발 (다리 사이 1~2픽셀 공백)
           └ 수비수: 한쪽 손 위치에 글러브 픽셀
           └ 타자:   몸 앞이나 어깨에 배트 픽셀
           └ 주자:   양손 비어있음
```

전체 실루엣: **달걀형 상단(모자) + 좁아지는 하단(다리)**

## 5. 납품 후 개발 측 처리

Canvas API로 런타임 팀 컬러 자동 교체:

```
field_defender.png  : #1E3A8A → 수비팀 colors[0]
field_batter_r.png  : #8B1A1A → 공격팀 colors[0]
field_batter_l.png  : #8B1A1A → 공격팀 colors[0]
field_runner.png    : #8B1A1A → 공격팀 colors[0]
```

팀 60개 모두 이 3개 파일만으로 커버됩니다.
