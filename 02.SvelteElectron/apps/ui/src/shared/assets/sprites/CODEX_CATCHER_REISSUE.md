# 포수 스프라이트 재제작 요청

## 문제점

기존 `catcher_set.png`, `catcher_receive.png` 는 포수의 **뒷모습**으로 제작되었습니다.
게임 시점이 **투수 뒤에서 홈플레이트 방향을 바라보는 구도**이기 때문에
포수는 카메라(투수)를 향해 앉아 있으므로 **정면이 보여야** 합니다.

## 참고 이미지

`apps/ui/src/shared/assets/match-demo/stadium-scene-concept.png`

이 이미지의 포수를 보면:
- **정면에서 보이는** 가슴보호대(chest protector) 앞면
- **정면에서 보이는** 마스크 앞면
- 쪼그려 앉은 자세, 카메라(투수) 쪽을 향함

## 재제작 요청

| 파일명 | 크기 | 포즈 |
|---|---|---|
| `catcher_set.png` | 96 × 112px | **정면** — 가슴보호대 앞, 마스크 앞, 쪼그려 앉아 글러브 무릎 위에 |
| `catcher_receive.png` | 96 × 112px | **정면** — 공 받으려 글러브 앞으로 뻗음 |

## 스타일 기준

`stadium-scene-concept.png` 포수와 동일한 품질·스타일.

- 치비 비율
- 다크 그레이/블랙 장비 (팀 컬러 교체 없음 — 고정색)
- PNG 투명 배경
- 픽셀아트 AI 생성 품질
