# 야구 게임 PNG 스프라이트 제작 의뢰서 (v2)

## 핵심 요구사항

기존 SVG 방식을 버리고 **PNG 이미지**로 전환합니다.
목표 품질: `stadium-scene-concept.png` 안에 이미 그려진 캐릭터와 **동일한 수준**.

---

## 필수 참고 — 스타일 정의 이미지

### ① `apps/ui/src/shared/assets/match-demo/stadium-scene-concept.png` (최우선)

이 이미지 안의 투수·포수·타자가 **목표 품질 기준**입니다.

| 캐릭터 | 특징 |
|---|---|
| **투수** | 화면 왼쪽 하단, 뒷모습, 흰 유니폼 + 네이비 소매/캡, 투구 자세 |
| **포수** | 홈플레이트 뒤, 뒷모습, 다크 장비 일체 (마스크·가슴보호대·정강이), 쪼그려 앉음 |
| **타자** | 오른쪽, 측면, 빨간 헬멧·유니폼, 배트 들고 대기 |

캐릭터 아트 특징:
- **치비 비율** (머리 크고 몸 짧음)
- 외곽선 선명, 명암 최소
- 선명한 색상 (채도 높음)
- **픽셀아트 스타일** 이지만 AI 이미지 생성 품질

### ② 추가 레퍼런스
- `apps/ui/src/shared/assets/match-demo/stadium-backdrop-concept.png`
  → 배경만 있는 버전 (캐릭터 없음). 최종적으로 이 배경 위에 스프라이트가 올라감
- 구글 검색: **"pixel art baseball chibi sprite transparent PNG"**
- 구글 검색: **"retro baseball game character sprite sheet"**

## 카메라 시점

경기 화면은 **포수 등 뒤에서 마운드 방향**을 보는 시점입니다.

| 캐릭터 | 시점 |
|---|---|
| 투수 | **뒷모습** — 등·어깨·뒷머리만 보임 |
| 포수 | **뒷모습** — 장비 뒤쪽이 보임, 쪼그려 앉음 |
| 타자 | **측면** — 오른쪽 면이 보임 (타자가 투수 방향으로 섬) |
| 심판 | **뒷모습** — 포수 바로 뒤에 서 있음 |

---

## 팀 컬러 처리

PNG는 CSS 변수를 지원하지 않으므로 **기본 색상으로 고정** 제작합니다.

| 캐릭터 | 유니폼 기본색 |
|---|---|
| 투수 | 흰 유니폼 + 네이비 소매·캡·번호 |
| 포수 | 다크 그레이/블랙 장비 |
| 타자 | 빨강 헬멧 + 빨강/흰 유니폼 |
| 심판 | 다크 네이비/블랙 |
