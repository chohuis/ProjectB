# 02.SvelteElectron

## Overview
- Desktop stack: `Svelte + Electron`
- Core logic: `packages/core`
- Shared contracts: `packages/contracts`

## Dev
```powershell
npm.cmd install
npm.cmd run dev
```

## Build
```powershell
npm.cmd run build
```

## Workspace
- `apps/ui`: UI
- `apps/desktop`: Electron shell
- `packages/core`: game engine core
- `packages/contracts`: DTO and IPC contracts
