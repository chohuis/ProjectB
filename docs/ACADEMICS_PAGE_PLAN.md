# Academics Page Plan

## Current Dummy Scope
- University section uses a fixed dummy major: `체육교육`.
- University metrics are shown as `학점`, earned credits, attendance, assignment, and warning risk.
- If the player does not attend university, the university segment stays locked.

## Planned Expansion (Post-Admission)
- At university admission, the player will choose one major from:
  - `스포츠과학`
  - `체육교육`
  - `스포츠경영`
  - `재활운동`
  - `스포츠심리`
- Major selection will later connect to training/performance modifiers.

## UI Notes
- Keep the current single-page structure with `고등학교/대학교` segment tabs.
- Maintain stage-gated navigation behavior:
  - `학업` tab visible only for high school / university stages.
