## 2. 표시 원칙

- **정보 전부 공개**(학교 선택) — 강호/배경 구분 없이 전력★·팀특성을 동일 형식으로([07_주인공_생성](../02_기획/07_주인공_생성.md) §3-2). 히든스탯(천재성·인성·성실함)은 이 화면에 **등장 자체를 안 함**(§5, 완전 비공개 랜덤).
- 능력치 초기값도 화면에 숫자로 안 보여줌 — 타입만 고르면 배경 배정(§4). "무명에서 시작" 톤.
- 타입별 시작 구종 후보가 달라짐(제구형만 패스트볼 밖 접근 — §6) → 스텝 6이 스텝 5에 연동.
