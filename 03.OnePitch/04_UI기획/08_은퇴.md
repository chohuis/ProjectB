# UI — 은퇴 화면 (엔딩)

> 상태: **구조 확정(초안) — 기존 문서에서 도출**
> 게임의 정체성 본체([05_히스토리_엔딩](../02_기획/05_히스토리_엔딩.md)). 은퇴 트리거 시 진입하는 종료 연출. **"인생은 점수로 판정하지 않는다"** — 등급·점수 없이 기록과 이야기로 응축.

## 1. 진입

- 은퇴 트리거 3종([05_히스토리_엔딩](../02_기획/05_히스토리_엔딩.md) §3): 자발적 / 노쇠·방출 압박 / 부상 강제. 진입 계기는 다르나 화면은 동일.

## 2. 구성 요소 ([05_히스토리_엔딩](../02_기획/05_히스토리_엔딩.md) §4)

| 요소 | 내용 |
|---|---|
| 서술형 커리어 내레이션 | 주요 시즌·사건·라이벌 관계를 엮은 이야기([06_서술_템플릿](../02_기획/콘텐츠/06_서술_템플릿.md) 조립) |
| 통산 기록 대시보드 | ERA·승패·탈삼진 등 + **최종 자산**([08_개인_재정](../02_기획/08_개인_재정.md) §4), **등급 매김 없이 순수 숫자** |
| 커리어 타임라인 그래프 | 시즌별 성적 곡선(무명→정상→황혼) — [기록 허브](03_기록.md)와 위젯 공유 |
| 하이라이트 모음 | **자동 선정**([05_히스토리_엔딩](../02_기획/05_히스토리_엔딩.md) §4-2, 플레이어 개입 없음) — 개인기록>우승결정전>라이벌전>국대>S등급, 데뷔전 고정 포함 |
| 은퇴식 연출 | 팬·동료·감독 반응(컷신 텍스트) — **명성 등급에 비례해 규모·문구 달라짐**([09_평가_시스템](../02_기획/육성코어/09_평가_시스템.md) §4-1) |
| 명예 요소 | 수상 목록·타이틀 히스토리 |

## 3. 이후 흐름 ([05_히스토리_엔딩](../02_기획/05_히스토리_엔딩.md) §4-1)

- 확인 후 **메인 메뉴 복귀**. "새로 시작" = 완전히 새 세이브 슬롯(회차 간 연결 없음).
- 기존 세이브는 남아 은퇴 화면·기록보관소를 언제든 다시 열람 가능.
