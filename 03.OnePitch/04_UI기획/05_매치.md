# UI — 매치 (프리게임 · 경기 · 결과요약)

> 상태: **구조 확정(초안)**
> 게임의 간판 화면. `game` PendingAction → 프리게임(모드 선택) → 경기 → 결과 요약.

## 결정 6 — 표현 모델 = 텍스트 중심 + 상시 경량 상황판 + 1구 조작 집중뷰

순수 텍스트(FM)도, 풀 궤적 애니(Flame)도 아닌 중간.

- **상시 경기 상황판**(경량 시각): 다이아몬드(주자 위치)·이닝·스코어·볼카운트·아웃. 항상 화면에 보임.
- **텍스트 진행**: 이닝/타석 서술([06_서술_템플릿](../02_기획/콘텐츠/06_서술_템플릿.md)). 볼 궤적 애니는 없음.
- **1구 조작 집중뷰**: 수동·클러치 순간에만 3×3 존 그리드 + 구종 선택 + 볼카운트.
- **렌더 엔진 = CustomPainter**([03_구조](../03_설계/03_구조.md) §8 해소) — 상황판·존 그리드 수준이라 Flame 불필요. 추후 궤적 애니를 원하면 Flame 재검토 여지만 남김.

## 1. 프리게임 — 모드 선택 (경기 시작 전 1회, 고정)

- 자동 / 수동 / 반자동 중 택1([07_매치_엔진](../02_기획/육성코어/07_매치_엔진.md) §3) — 경기 내내 고정.
- 등판 상황(선발/구원)·상대팀·구장(파크팩터)·날씨 요약 표시.

## 2. 상시 경기 상황판 (3모드 공통)

- 다이아몬드 + 주자 · 이닝 · 스코어 · B-S-O 카운트. CustomPainter로 간결하게.
- 주인공 라이브상태(피로·컨디션) 소형 표시 — 교체 판단 참고.

## 3. 모드별 진행

| 모드 | 진행 방식 |
|---|---|
| **자동** | 상황판 + 요약 텍스트로 관전, 조작 없음. 빠른 진행(스킵 가능) |
| **수동** | 매 타석 1구 조작 집중뷰(§4)에서 직접 투구 |
| **반자동** | 평상시 요약 텍스트, **결정적 순간([07_매치_엔진](../02_기획/육성코어/07_매치_엔진.md) §4)에만 집중뷰로 격상** — 위기·기록근접·라이벌매치업 |

- **모드 무관 동일 기록**([07_매치_엔진](../02_기획/육성코어/07_매치_엔진.md) §12) — 어느 모드든 결과·평가 동일, 조작은 표시·개입 수준만.

## 4. 1구 조작 집중뷰

- **구종 선택**: 보유·습득 구종만 버튼([05_구종_시스템](../02_기획/육성코어/05_구종_시스템.md)) — 마스터리 단계 표시.
- **코스 선택**: 스트라이크존 3×3 그리드(9분할, [07_매치_엔진](../02_기획/육성코어/07_매치_엔진.md) §5).
- **볼카운트 진행**: 매 구 볼/스트라이크/파울/인플레이/사구 판정 누적 표시.
- 입력 흐름(구현 상세): 구종 → 코스 셀 → 투구. 결과는 상황판·텍스트에 반영.
- 판정식은 엔진(은닉, [05_밸런스](../03_설계/05_밸런스.md) B) — UI는 결과만 받음.

## 5. 감독 개입 — 투수 교체

- **자동·반자동**: 감독 AI가 판단([07_매치_엔진](../02_기획/육성코어/07_매치_엔진.md) §8), 결과 통지.
- **수동**: 이닝 종료·위기마다 "교체할까요?" 프롬프트 — 강행/수용이 [04_프로_커리어](../02_기획/04_프로_커리어.md) 감독 신뢰(관계도)에 반영.

## 6. 반자동 격상 전환 연출

- 요약 진행 중 결정적 순간 도달 → 집중뷰로 전환. 명확한 알림(배너/전환)로 "지금부터 직접"임을 알림([07_매치_엔진](../02_기획/육성코어/07_매치_엔진.md) §13 열린세부 해소 방향).

## 7. 경기 종료 → 결과 요약 화면

- [09_평가_시스템](../02_기획/육성코어/09_평가_시스템.md) §5-5: 경기 중 평가는 계산돼 있고 **종료 후 요약화면에 몰아서** 표시(흐름 안 끊음).
- 구성: 최종 스코어·주인공 기록(이닝·실점·탈삼진 등)·**S~D 등급**·감독/스카우트 코멘트·하이라이트. 상세는 [기록 허브](03_기록.md) 경기 로그에 영구 보존.
- **경기 중 저장 불가**([07_매치_엔진](../02_기획/육성코어/07_매치_엔진.md) §12) — 이 요약 확정 시점에 커밋.
