# UI — 내 선수 허브 (상태 · 훈련 · 재정)

> 근거: [00_개요](00_개요.md) 결정 1·4 · [01_선수_능력치](../02_기획/육성코어/01_선수_능력치.md) · [05_구종_시스템](../02_기획/육성코어/05_구종_시스템.md) · [06_훈련_시스템](../02_기획/육성코어/06_훈련_시스템.md) · [08_개인_재정](../02_기획/08_개인_재정.md) · [09_평가_시스템](../02_기획/육성코어/09_평가_시스템.md) §4-1(명성)

> 4허브 중 첫 번째, 기본 랜딩. 3탭 구조.

## 1. 상단 요약 바 (3탭 공통 고정)

- 이름 · 나이 · 소속팀 · 역할(선발/불펜/마무리/스윙맨, [04_프로_커리어](../02_기획/04_프로_커리어.md)) · **명성 라벨**(무명~레전드 5단계, [09_평가_시스템](../02_기획/육성코어/09_평가_시스템.md) §4-1) · 좌/우완.
- 전역 표시 언어([00_개요](00_개요.md) §3) 적용 — 명성은 라벨, 수치 비노출.

## 2. 상태 탭 (기본)

- **능력치 9개** — 피지컬·기술·멘탈 **3카테고리로 그룹**([01_선수_능력치](../02_기획/육성코어/01_선수_능력치.md) §1). 각 값은 20~80 숫자 + **값에 따라 색 변함**(결정 4).
- **구종 목록** — 보유 구종별 이름 + 마스터리 5단계 라벨(습작~필살기, [05_구종_시스템](../02_기획/육성코어/05_구종_시스템.md) §2). 미보유/미습득 구종은 회색 또는 비표시.
- **라이브상태** — 피로·컨디션·사기 **게이지 바 3개**(결정 4). 매주 갱신.
- **히든스탯(천재성·인성·성실함) 없음** — 화면에 존재 자체를 드러내지 않음([01_선수_능력치](../02_기획/육성코어/01_선수_능력치.md) §4). 코치 피드백·이벤트로만 간접 짐작.

## 3. 훈련 탭

[06_훈련_시스템](../02_기획/육성코어/06_훈련_시스템.md)의 "지속 설정"을 조작하는 화면 — [04_게임루프](../03_설계/04_게임루프.md) §3에 따라 상시 접근(PendingAction 아님). [03_메시지_알림](../02_기획/콘텐츠/03_메시지_알림.md) §2-1 "주간 훈련 슬롯 배분 화면"이 이 탭.

- **능력치 슬롯**: 주슬롯 1 + 보조슬롯 2 = 3개 → 9개 능력치 중 배분.
- **구종 슬롯**: 1개 → 신규 습득 or 기존 마스터리업.
- **강도 다이얼**: 슬롯별 약/보통/강 — **등판 주엔 "강" 비활성**([06_훈련_시스템](../02_기획/육성코어/06_훈련_시스템.md) §4-1). 등판 예정이면 UI가 "강" 잠금 표시.
- **개인 트레이너**: 3분야(피지컬/기술/멘탈) × 2단계(일반/전문가) 중 1개 활성/해제([08_개인_재정](../02_기획/08_개인_재정.md) §3-2). 팀 코치 공백을 메우는 선택이라, 소속팀 코치 배치 상태를 함께 참조 표시하면 유용.
- "지속 설정"이라 매주 다시 짤 필요 없음 — 바꿀 때만 조작.

## 4. 재정 탭

[08_개인_재정](../02_기획/08_개인_재정.md) — 완전 자동 관리라 가계부·예산배분 UI 없음. **잔액 + 조회**만.

- **현재 잔액**(세후).
- **최근 명세**: 수입(연봉·계약금·스폰서)·지출(사치품·트레이너·투자)·세금 세전/세후 내역 리스트.
- **개인 트레이너 구독 상태**·**투자 성향**(저축/공격) 표시 — 조작은 각 경로(훈련 탭 / 시즌말 투자 알림)에서, 여기선 현황만.
