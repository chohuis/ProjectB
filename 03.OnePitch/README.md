# OnePitch — 정식버전 기획·설계 문서

투수 인생 시뮬레이션 "투수 키우기"의 **정식버전(OnePitch)** 문서 저장소.
프로토타입 `02.SvelteElectron`을 분석·계승해 정식 품질로 재구축하는 것을 목표로 한다.

## 계보

```
00.OneF (WPF 원형 프로토 — 원 기획 7문서)
   └→ 02.SvelteElectron (Svelte+Electron+Rust DLL 프로토타입, Lite 재기획 R0~R5)
        └→ 03.OnePitch (정식버전) ← 현재
```

## 설계 확정 요약 (2026-07-09)

- **스택**: Flutter(Dart UI) + Rust 엔진(flutter_rust_bridge FFI) · **PC + Android** · web 제외
- **계층 계약**: 모든 계산은 Rust 엔진에 숨겨 결과만 반환 · UI는 렌더+입력만 · DB 직접접근 0
- **데이터 3곳(전부 엔진 뒤)**: `content.db`(콘텐츠, 직접저작+AI import) · `slot.db`(세이브) · Rust 상수(은닉 밸런스, D4-2)
- **견고성**: 처음부터 단계별 세이브 마이그레이션 + HMAC 변조감지 · 로컬 자동백업
