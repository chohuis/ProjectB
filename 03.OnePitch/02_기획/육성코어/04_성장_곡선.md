# 성장 곡선 (XP · 노쇠)

> 근거: [육성코어 개요](00_개요.md) · [01_선수_능력치](01_선수_능력치.md) · [02_스태프_능력치](02_스태프_능력치.md) · [04_프로_커리어](../04_프로_커리어.md) Phase5
> 상태: **구조 확정(초안)** · 정확한 계수는 스탯 스케일 확정 후

## 1. 두 곡선의 합성

```
전체 커리어 곡선 = 상승기(XP→스탯) + 하락기(나이→노쇠)
```

- 커리어 초반~중반은 상승기가 지배, 후반은 하락기가 지배 — 둘은 언제나 동시에 작동하며 상승분과 하락분의 순합이 그 해 스탯 변화.

## 2. 상승기 — XP → 스탯

- **XP 획득원**: ① 주간 훈련(구체 배분은 [06_훈련_시스템](06_훈련_시스템.md)) ② 경기 실전 경험 ③ 이벤트 보상([콘텐츠/02_이벤트](../콘텐츠/02_이벤트.md) §3). 단 **이벤트 XP는 특정 능력치를 지정해 부여**(예: 멘탈 관련 이벤트 → 침착함 XP) — 어디에나 쓰는 범용 자유 XP가 아님.
- **변환 시점 = 매주**(훈련 사이클과 동일 주기): XP가 누적되다가 임계값을 돌파하면 **그 주에 즉시 스탯 +1** — 별도 배치 없이 주간 단위로 바로 반영.
- **점진적 임계값 곡선**: 스탯이 높을수록 다음 +1에 필요한 XP가 늘어남(정확한 계수는 스탯 스케일 확정 후) — 초반엔 빠르게, 후반엔 더디게 성장.
- **천재성(히든)이 곡선을 좌우**: 기울기(성장 속도)와 상한(재능 캡) 둘 다에 영향. 천재성이 높으면 같은 훈련으로 더 빠르게, 더 높은 한계까지 성장.

## 3. 하락기 — 나이 → 노쇠 (카테고리별 비대칭)

핵심 설계: **피지컬은 먼저 꺾이고, 기술·멘탈은 나이 들어도 유지·성장 가능**.

| 카테고리 | 하락 양상 |
|---|---|
| **피지컬**(구속·체력·회복력 / 파워·스피드·체력) | 20대 초중반 정점 → 이후 하락 시작. **하락 시점·속도는 개인별 가변**(§4) |
| **기술**(제구·구위·경기운영 / 컨택·선구안·수비) | 30대까지도 유지·성장 가능 — 경험이 누적되는 영역 |
| **멘탈**(클러치·침착함·리더십) | 나이와 거의 무관, 완만한 상승도 가능 |

- **서사 함의**: 파이어볼러(피지컬 의존)는 일찍 저물고, 그 시점에 **제구·경기운영으로 버티는 기교파로 전향**하는 흐름이 시스템에서 자연히 발생 — [04_프로_커리어](../04_프로_커리어.md) Phase5 "구위 하락 → 기교파 전환"과 일치.

## 4. 피지컬 하락 시점 — 개인별 가변 요인

하락 시작 나이·속도가 아래 요인들로 조정된다:

| 요인 | 영향 |
|---|---|
| 체력 스탯(투수는 회복력도) | 높을수록 하락 시작 지연 |
| 부상 이력 | 누적될수록 조기 하락 → [08_부상_시스템](08_부상_시스템.md)과 연동 |
| 컨디셔닝코치 품질 | 좋은 관리를 받으면 하락 지연 — [02_스태프_능력치](02_스태프_능력치.md) 코치 매칭 |
| 히든스탯 성실함 | 자기관리 잘하는 선수는 하락 지연 |
| 팀특성 ①철학 | 부상방지/재활특화 소속=하락 지연, 스파르타(혹독훈련) 소속=조기 하락 위험 — [팀 특성 시스템](../리그팀/05_팀_특성_시스템.md) |

→ "예측 어려움 + 관리를 잘하면 늦게 꺾인다"는 서사 여지를 개인별 가변으로 확보.
