## 1. 원칙 — 상대평가(기대치 대비)

같은 기록(예: 7이닝 2실점)이라도 **상황에 따라 다른 등급**을 받는다. 절대 수치(ERA·탈삼진 등)가 항상 같은 등급으로 매겨지지 않음 — 약팀 상대 완봉보다 강팀 상대 선전이 더 높게 평가될 수 있다.

## 2. 기대치 4요소

| 요소 | 방향 |
|---|---|
| **상대 타선 수준** | [03_팀_전력_공식](03_팀_전력_공식.md) 타선 전력이 강할수록 기대치↑(잘 던져도 덤덤히 봄), 약할수록 기대치↓ |
| **등판 상황(역할)** | 역할별 기대 기준이 다름 — 선발=이닝수 기대, 마무리=세이브상황 마무리 기대 |
| **경기 중요도** | 평범한 리그경기보다 포스트시즌·결승전 같은 큰 무대에선 기대치 자체가 상승(대신 잘하면 보상도 큼) |
| **본인의 평소 실력 수준** | 에이스급 투수는 기대치가 높아 평범한 피칭도 낮게 평가될 수 있음 — **가중치는 가볍게**(다른 3요소의 보조 보정 수준). 너무 가혹하면 에이스가 S를 못 받는 역설 발생 → §5 열린세부에 명시 |

## 3. 등급 6단계 (S/A/B/C/D/F)

| 등급 | 의미 |
|---|---|
| **S** | 압도적 역투 — 기대치를 크게 뛰어넘음 |
| **A** | 우수한 호투 |
| **B** | 평균 이상, 무난 |
| **C** | 평균 이하, 아쉬움 |
| **D** | 부진 |
| **F** | 완전 붕괴·참패 |

## 4. 등급의 효과 (4가지)

| 효과 | 연결 시스템 |
|---|---|
| **감독 신뢰도** | [02_스태프_능력치](02_스태프_능력치.md) 신뢰형성력 + [04_프로_커리어](../04_프로_커리어.md) Phase3 역할배정에 직결 — 좋은 등급 누적이 신뢰를 쌓음 |
| **주목도** | 경기별 등급이 누적되어 [02_아마_고교](../02_아마_고교.md) §D "주목도=리그성적+대회임팩트+마일스톤" 공식의 **"리그 성적" 핵심 입력값**이 됨. 드래프트 가치·스카우팅 평가로 이어짐 |
| **사기** | [01_선수_능력치](01_선수_능력치.md) §5 라이브상태 사기에 직접 가감 |
| **커리어 기록** | [07_매치_엔진](07_매치_엔진.md) career_history·game_log에 경기별 등급도 함께 영구 기록 — 히스토리 엔딩의 재료 |

## 4-1. 명성(팬덤·스타성) — 주목도와 분리된 파생값 (대화 설계 2026-07-14)

**주목도**(§4, [02_아마_고교](../02_아마_고교.md) §D)는 "스카우트가 보는 실력 평가"고, **명성**은 "대중이 아는 화제성"이다 — 실력이 그대로 인기가 되지 않을 수 있어 별도 개념으로 둔다(논란의 캐릭터가 인기는 많은 경우 등).

- **저장하지 않는 파생값**: [03_팀_전력_공식](03_팀_전력_공식.md)의 "실전력"과 동일 원칙 — slot.db에 새 필드를 만들지 않고 매번 재계산. `명성 = f(주목도 누적, 최근 이벤트·뉴스 노출 빈도, 국대·타이틀 등 하이라이트 마일스톤 가산)`.
- **라벨 버킷(5단계)**: 무명 → 유망주 → 주목받는 선수 → 스타 → 레전드 — [01_선수_능력치](01_선수_능력치.md) §7 "화면은 거칠게" 원칙 재사용, 숫자 비노출.
- **영향 지점**:
  - [콘텐츠/05_뉴스_미디어](../콘텐츠/05_뉴스_미디어.md) — 소속팀 소식 노출 상세도·인터뷰 요청 이벤트([콘텐츠/02_이벤트](../콘텐츠/02_이벤트.md) 확률형 플레이버) 빈도가 명성 등급에 비례.
  - [05_히스토리_엔딩](../05_히스토리_엔딩.md) §4 은퇴식 연출 — "팬·동료·감독의 반응" 컷신 강도가 명성 등급에 비례(레전드급 은퇴 vs 무명 은퇴는 같은 템플릿이라도 팬 반응 문구·규모가 달라짐).
  - **스폰서·CF 등 금전적 연결은 이월** — [06_시장_계약](../06_시장_계약.md)과 함께 개인 재정 스코프 결정 후 구체화(지금은 비화폐적 서사 반응까지만).

### 5-1. 3단계 평가 주기

| 주기 | 시점 | 역할 |
|---|---|---|
| **경기 단위** | 매 등판 종료 직후(계산) | S~D 산출 → 신뢰도·사기 즉각 반영, game_log에 개별 기록 |
| **월 단위** | 매월 말 | 그 달 경기별 등급을 종합한 "이달의 페이스" — 사기 보정·팬/미디어 반응 이벤트의 재료 |
| **시즌 단위** | 시즌 종료 | 전체 누적 → **주목도 최종 확정**([02_아마_고교](../02_아마_고교.md) §D "리그성적" 요소), 연봉재협상 기초자료, **방출 종합지표 핵심 입력**([06_시장_계약](../06_시장_계약.md) §1), 시상 후보([05_히스토리_엔딩](../05_히스토리_엔딩.md) 수상로그) |

### 5-2. 평가 대상 — 주인공만

NPC 투수·타자는 경기 데이터(스탯)는 쌓이지만 **S~D 등급은 부여하지 않음** — 시스템 단순화.

- **NPC 로테이션·역할 판단**: 등급 대신 **능력치(구속·제구·구위·경기운영 등) + 누적 성적**([07_매치_엔진](07_매치_엔진.md)이 쌓은 스탯)만으로 감독 AI가 서열을 매겨 로테이션·역할을 편성한다 — [04_프로_커리어](../04_프로_커리어.md) Phase3, [03_팀_전력_공식](03_팀_전력_공식.md) 하위지표(선발진·불펜)와 동일 데이터를 사용.
- S~D 등급은 오직 주인공만의 **"서사적 피드백 렌즈"** — 주인공은 스탯·성적에 더해 등급까지 받아 감독 신뢰·주목도·사기에 반영되고, NPC는 스탯·성적만으로 조용히 처리된다. 같은 자리를 두고 경쟁하지만 등급 연산은 주인공 한 명에게만 돌아 시스템이 가벼워진다.

### 5-3. 최소 등판 기준

| 역할 | 평가 대상 최소 기준 |
|---|---|
| 선발 | **1이닝 이상** 소화 |
| 구원·마무리 | **1아웃 이상** 기록 |
| 미달(부상강판 등) | 등급 없이 "미완료"로만 기록 — F등급(완전붕괴)과는 구분(실패가 아니라 데이터 부족) |

### 5-4. 역할별 계산 트리거 (경기 단위)

| 역할 | 계산 시점 |
|---|---|
| 선발 | **강판(교체) 시점** |
| 구원(중간계투) | 본인이 마운드 내려올 때 |
| 마무리 | **경기 종료 시**(세이브·블론세이브 여부까지 포함해 판정) |

### 5-5. 결과 표시 시점 — 경기 종료 후 요약화면

- **계산은 §5-4 트리거 시점에 이미 완료**되지만, **화면 표시는 경기 전체가 끝난 뒤 요약화면에 몰아서** — 즉각 팝업으로 경기 흐름을 끊지 않음.
- 상세 설명·과거 기록은 [05_히스토리_엔딩](../05_히스토리_엔딩.md) 상시 기록보관소에서 추가 확인 가능.
