# 구종 시스템 (투수) · 타격 유형 태그 (타자)

> 근거: [육성코어 개요](00_개요.md) · [01_선수_능력치](01_선수_능력치.md)
> 상태: **구조 확정(초안)**
> 구종은 투수 전용 개념(타자 대응 불가) — 타자는 §3의 라이트 태그로 대신함(육성코어 개요 대화, 2026-07-09).

## 1. 구종 카탈로그 (10종, 3계열×3 + 특수구 1)

| 계열 | 구종 |
|---|---|
| **패스트볼류** | 포심 패스트볼 · 투심 패스트볼 · 커터 |
| **브레이킹볼류** | 슬라이더 · 커브 · 스위퍼 |
| **오프스피드류** | 체인지업 · 포크볼 · 싱커 |
| **특수구** | **너클볼** — 기존 3계열에 안 속하는 이질적 구종. 회전이 아니라 무회전으로 승부하는 완전히 다른 메커니즘이라 단독 분류 |

- [01_선수_능력치](01_선수_능력치.md)의 "구위(무브먼트)"는 이 카탈로그와 안 겹치는 **상위 개념**(전반적 구위) — 구종 하나하나의 개별 마스터리는 여기서 별도 관리.

## 2. 마스터리 5단계 (새 명명)

| 단계 | 이름 | 상태 |
|---|---|---|
| 1 | **습작** | 막 익히기 시작, 불안정 |
| 2 | **연마** | 다듬는 중, 기복 있음 |
| 3 | **실전** | 실전 투입 가능, 안정적 |
| 4 | **주무기** | 확실한 무기, 신뢰도 높음 |
| 5 | **필살기** | 최상급, 그 선수의 시그니처 구종 |

- 구종마다 독립적으로 5단계를 밟는다 — 한 선수가 어떤 구종은 필살기, 어떤 구종은 습작 단계일 수 있음.

## 3. 시작·습득

- **시작 구종**: 신인은 **1~2개**만 갖고 시작(예: 포심 단독, 또는 포심+체인지업류) — 리그·팀 생성 규칙에 따라 조정. 적은 시작 레퍼토리가 "레퍼토리를 넓혀가는 성장 서사"의 여백이 됨.
- **신규 습득**: 훈련 프로그램의 "신규 구종 연마" 슬롯 투자로 시작([06_훈련_시스템](06_훈련_시스템.md)에서 구체화).
- **습득 조건**: 관련 능력치(제구·구위) 임계 이상 + 우연 이벤트로 발견 가능("불펜에서 각도가 좋은 슬라이더를 우연히 찾았다" 류).
- **코치별 구종 전문 목록(소프트 보너스)**: 각 투수코치([02_스태프_능력치](02_스태프_능력치.md))는 "가르칠 수 있는 구종" 목록(카탈로그 10종 중 일부)을 보유. 해당 코치가 팀에 있으면 그 구종 습득 속도·성공률이 크게 상승. **하드락 아님** — 전문 코치 없이도 느리게는 배울 수 있음(궁핍한 팀이 영구히 특정 구종을 못 배우는 가혹함 방지), 대신 팀 선택·이적에 실질적 무게가 실림("이 학교엔 스위퍼 가르칠 코치가 없네").
- **소요 기간**: 5단계를 다 밟는 데 여러 주~시즌 단위 시간 소요.

## 4. 효과 (구조만 — 정확한 수치는 매치 엔진에서)

- **개별 구종 위력**: 그 구종의 마스터리 단계가 실전 결과(피안타율·헛스윙 유도 등)에 직결.
- **레퍼토리 조합 효과**: 보유 구종 수·계열 다양성이 타자 교란(예측 어려움)에 기여 — 같은 계열만 여러 개보다 **3계열 골고루** 보유가 유리하도록 설계.
- **역할 적합성**: 특정 구종이 특정 상황에 강함(예: 싱커=땅볼 유도로 빠른 이닝 소화 → [01_선수_능력치](01_선수_능력치.md) "경기운영" 스탯과 시너지, 포크볼=헛스윙 유도로 삼진형).

## 5. 타격 유형 태그 (타자, 라이트)

전용 능력치 시스템은 만들지 않음 — **서사·해설용 태그**만 부여. 타자 능력치([01_선수_능력치](01_선수_능력치.md) §3)에서 지배적인 스탯 조합으로 자동 태깅.

| 태그 | 판정 근거(지배 스탯) |
|---|---|
| 파워형 | 파워 압도적 |
| 컨택형 | 컨택 압도적, 삼진 적음 |
| 스프레이형 | 컨택+파워 균형, 전방향 타격 |
| 스피드형 | 스피드 압도적, 내야안타·도루 위협 |
| 인내형 | 선구안 압도적, 출루형 |
| 올라운드형 | 특정 스탯 쏠림 없이 고르게 높음 |

- 태그는 매 시즌 스탯 변화에 따라 **재판정**될 수 있음(고정 라벨 아님).
- 용도: 뉴스·해설 텍스트("전형적인 파워형 거포"), 스카우팅 리포트, 팀 특성 시스템 이벤트 트리거와 연계 가능.

## 6. 열린 세부 (다음 결정)

- [ ] 구종별 정확한 습득 조건 수치(능력치 임계값) — 스탯 스케일 확정 후
- [ ] 레퍼토리 다양성 보너스 정확한 수치
- [ ] 리그별(고교~프로) 시작 구종 수 차등 여부
- [ ] 타격 유형 태그의 재판정 주기(매 시즌? 누적 데이터 기반?)
