# 스태프 능력치 체계 (감독 · 코치 · 구단주)

> 근거: [육성코어 개요](00_개요.md) · [01_선수_능력치](01_선수_능력치.md) · 대화 설계(2026-07-09)
> 상태: **확정(초안)**
> 선수(운동선수)와 달리 스태프는 **각자 다른 것을 통제하는 역할**이라, 피지컬/기술/멘탈 틀을 재사용하지 않고 **역할별 기능 축**으로 설계.
> **수치 스케일**: [01_선수_능력치](01_선수_능력치.md) §7 표준(20~80 범위, 내부연속치, 화면 3단위 반올림)을 동일 적용.

## 1. 감독 — 역할배정·신뢰·전술

| 능력치 | 무엇 |
|---|---|
| 전술력 | 경기 중 작전·운영 판단 |
| 신뢰형성력 | 선수와 신뢰 쌓는 속도 — [04_프로_커리어](../04_프로_커리어.md) "감독 배정(신뢰형)" 메커니즘에 직결 |
| 육성안목 | 잠재력 판별력 — 선수 히든스탯([01_선수_능력치](01_선수_능력치.md) §4)의 힌트를 줄 수 있는 존재 |
| 카리스마 | 팀 사기·리더십 파급력 |

- **감독 성향 ↔ 팀특성 ①철학 연동**(대화 설계 2026-07-14): 새 계수를 만들지 않고 [콘텐츠/01_캐릭터](../콘텐츠/01_캐릭터.md) §3 "역할·소속 가중 랜덤"을 감독에도 그대로 적용 — 감독도 결국 NPC이므로, 소속팀 철학(스파르타·데이터중심 등)에 어울리는 성격 3슬롯이 나오기 쉽게 가중된다. 스파르타 팀 감독은 승부사·완벽주의 성향이 자주 나오는 식.

## 2. 코치 — 단일 역할, 능력치 프로필이 보직을 결정

코치는 고정 보직 태그 없이 **하나의 엔티티 타입**. 6개 전문영역 능력치 프로필 중 **최고값 영역**이 그 코치의 적성 보직을 암시한다.

### 2-1. 능력치 (전문영역 6 + 종합 1 = 7개)

| 능력치 | 대응 보직 | 무엇 |
|---|---|---|
| 투수 지도력 | 투수코치 | 투수 육성 전문. 부가 속성으로 **"가르칠 수 있는 구종" 목록**([05_구종_시스템](05_구종_시스템.md) 카탈로그 10종 중 일부) 보유 — 해당 구종 습득에 소프트 보너스 |
| 타격 지도력 | 타격코치 | 타격 육성 전문 |
| 주루 지도력 | 주루코치 | 타자 "스피드" 성장 + 도루·베이스러닝 판단 전담 |
| 컨디셔닝 | 컨디셔닝코치 | 체력·부상관리 |
| 멘탈 코칭 | 멘탈코치 | 심리 코칭 |
| 스카우팅 안목 | 전력분석관 | 유망주 발굴·전력분석 |
| **종합 지도력** | (전 보직 공통) | 전문영역과 별개로 전체 효율 배수 |

### 2-2. 보직 6종

투수코치 · 타격코치 · 주루코치 · 컨디셔닝코치 · 멘탈코치 · 전력분석관

### 2-3. 배정 규칙

- **적성 배치**: 코치의 6개 전문영역 중 최고값 영역 보직에 배치하면 전문 보너스 적용.
- **미스매치 배치**: 적성과 다른 보직에 배치 시 전문 보너스 없음, 종합지도력만 적용(효율 손실).
- **겸직**: 슬롯이 비면 감독 또는 다른 코치가 대신 수행 — **비적성 페널티**(그 영역 효율↓).
- **복수배치**(대화 설계 2026-07-14): 같은 보직에 코치를 여러 명(예: 투수코치 선발전담+불펜전담) 배치할 수 있음 — 새 규칙 불필요, 위 적성배치·미스매치 원칙을 각 코치에게 개별 적용할 뿐. 부유한 팀([팀_특성_시스템](../리그팀/05_팀_특성_시스템.md) §2-4)일수록 이 여유가 생김.

### 2-4. 슬롯 규모 (리그레벨 × 팀자원 가변)

| 조건 | 코치 슬롯 | 양상 |
|---|---|---|
| 프로 + 부유 | 6~8명 | 6보직 풀세트, 일부 보직 복수배치(투수코치 선발전담+불펜전담) 가능 |
| 프로 + 궁핍 | 3~4명 | 일부 보직 공백 or 겸직 |
| 독립 | 2~3명 | 핵심 보직(투수·타격) 우선, 나머지 감독 겸직 |
| 대학 | 2~3명 | 독립과 유사 |
| 고교(선택교/강호) | 1~2명 | 최소 배치 |
| 고교(배경·궁핍) | 0~1명 | 감독이 사실상 전부 겸직 — 육성 편차 발생 지점 |

- 궁핍한 팀일수록 코치 슬롯이 비고 겸직 페널티가 쌓임 — [팀 특성 시스템](../리그팀/05_팀_특성_시스템.md) "②자원" 축이 실질적 육성 격차로 이어지는 경로.

## 3. 구단주 — 재정·투자 성향

| 능력치 | 무엇 |
|---|---|
| 재력 | 팀특성 "②자원" 축을 실제로 만드는 근원 |
| 투자성향 | 공격적 영입 vs 내실 다지기 |
| 인내심 | 리빌딩 감내력 — 성적 부진 시 감독 경질 압박 등 서사에 영향 |

## 4. 총계

- 감독 4개 · 코치 7개(전문6+종합1) · 구단주 3개.
