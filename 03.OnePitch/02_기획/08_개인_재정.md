# 개인 재정 (수입·지출)

> 근거: [06_시장_계약](06_시장_계약.md)(수입원 전부 재사용) · [09_평가_시스템](육성코어/09_평가_시스템.md) §4-1(명성 — 스폰서 연결) · [콘텐츠/01_캐릭터](콘텐츠/01_캐릭터.md) §3(성격 3슬롯) · [05_히스토리_엔딩](05_히스토리_엔딩.md) · 대화 설계(2026-07-14)
> 상태: **구조 확정(초안)** · 정확한 세율 구간·보너스 수치는 스탯 스케일 확정 후
> 목적: 지금까지 돈은 [06_시장_계약](06_시장_계약.md)으로 **버는 것만** 있고 **쓰는 곳**이 없었다 — 순수 누적 스코어에서 벗어나 선택의 무게를 갖는 소비처를 만든다.

## 1. 원칙

- **수입원은 전부 기존 재사용** — 연봉·계약금·인센티브·FA는 [06_시장_계약](06_시장_계약.md) 그대로. 새 수입 메커니즘은 스폰서/CF(§1-1) 하나만 추가.
- **관리 UI = 완전 자동** — 가계부·예산배분 화면 없음. 상시 잔액만 표시([00_게임_개요](00_게임_개요.md) §3 "FM형 관리" 원칙 — 화면은 거칠게). 지출은 이벤트 선택 또는 지속 구독 옵션 활성화로만 발생.
- **소비처는 3종**: 사치품(1회성, §3-1) · 개인 트레이닝(지속 구독, §3-2) · 투자(시즌 단위 이지선다, §3-3).

### 1-1. 스폰서·CF — 명성 기반 신규 수입원

- [09_평가_시스템](육성코어/09_평가_시스템.md) §4-1 명성 등급이 높을수록 스폰서 제안 이벤트([콘텐츠/02_이벤트](콘텐츠/02_이벤트.md) 확률형 플레이버)가 더 자주, 더 좋은 조건으로 발생 — §4-1에서 "이월"해뒀던 부분을 여기서 연결.
- **4개 카테고리**: 용품(장비·운동화 등) / 음료·식품 / 지역상점(연고 지역 기반) / 디지털·게임. 명성 등급이 낮으면 지역상점·용품급만, 등급이 오를수록 전국단위 브랜드·음료/디지털 대형 CF까지 등장 범위가 넓어짐 — 명성 성장을 체감하게 하는 장치.
- 계약금과 마찬가지로 목돈 유입 — 정확한 금액은 스탯 스케일 확정 후.

## 2. 세금 — 간단 누진(2~3구간, 자동 공제)

- 연봉·계약금·스폰서 수입 등 모든 유입에 **수령 시점에 자동 공제** — 잔액에는 항상 세후 금액만 반영. 플레이어가 직접 신고·관리하지 않음(완전 자동 원칙).
- **저연봉 구간 < 중간 구간 < 고연봉 구간** 순으로 세율 상승 — 실제 소득세처럼 "많이 벌수록 비율도 오른다"는 사실감을 준다.
- 로그(내역)에는 "세전/세후" 명세가 남아 [05_히스토리_엔딩](05_히스토리_엔딩.md) 계약·이력 로그에서 확인 가능.
- 정확한 구간 경계값·세율은 스탯 스케일 확정 후 — [01_선수_능력치](육성코어/01_선수_능력치.md) §7 수치 표준과 함께 확정.

## 3. 지출 카테고리

### 3-1. 사치품·라이프스타일 (1회성, 이벤트 기반)

- 상시 쇼핑 메뉴 없음 — FA 대박 계약·시즌 활약 등 **성과 이후에만** "돈을 어떻게 쓸까?" 류 이벤트로 등장([콘텐츠/02_이벤트](콘텐츠/02_이벤트.md) "확률 기반(플레이버)" 트리거 유형 재사용).
- **소비 대상 = 자신 + 동료·스태프**: 차·시계 같은 자기 소비뿐 아니라 "후배에게 장비 선물", "팀 회식 한턱" 같은 선택지도 사치품 카테고리 안에 존재 — 이 경우 재정 지출에 더해 해당 NPC와의 [콘텐츠/01_캐릭터](콘텐츠/01_캐릭터.md) §5 관계도도 소폭 상승. 자신을 위한 소비는 관계도 효과 없이 §3-1 기본 효과(아래)만.
- **명성 영향은 성격 3슬롯에 따라 방향이 갈린다**([콘텐츠/01_캐릭터](콘텐츠/01_캐릭터.md) §3 ②경쟁성향) — 새 계수 없이 기존 성격 태그 재사용:
  - **야심가**: 화려한 소비가 "스타성"으로 읽혀 명성 소폭 **상승**.
  - **안분**: 같은 소비가 "과시"로 읽혀 명성 소폭 **하락**.
  - 나머지 성향(승부사·완벽주의·헌신형)은 중립(영향 없음).
- 능력치·훈련 효율에는 영향 없음 — 순수 서사+명성(+관계도) 연동, [콘텐츠/04_업적](콘텐츠/04_업적.md) "순수 명예, 게임플레이 효과 없음" 원칙과 결이 같음.

### 3-2. 개인 트레이닝·재활 투자 (지속 구독형, 3전문분야 × 2단계)

[02_스태프_능력치](육성코어/02_스태프_능력치.md)의 코치 전문영역 구조를 개인 고용 버전으로 축소 재사용 — 팀 코치가 빈 영역을 정확히 어떤 유형으로 메울지 플레이어가 고른다.

| 전문 분야 | 영향 스탯 | 비고 |
|---|---|---|
| 피지컬 트레이너 | 구속·체력·회복력 | [08_부상_시스템](육성코어/08_부상_시스템.md) 회복 속도 가속도 이 분야가 겸함 |
| 기술 코치 | 제구·구위·경기운영 | |
| 멘탈 코치 | 클러치·침착함·리더십 | |

- **2단계(일반/전문가)**: 전문가가 비용이 크고 보너스도 큼. 한 번에 **1개 분야만** 활성화 가능(구독형 — 여러 분야 동시 고용은 비용 부담이 커 기본 스코프 밖, 확장 여지는 열어둠).
- 훈련 효율 보너스는 [06_훈련_시스템](육성코어/06_훈련_시스템.md) 강도 다이얼과 별개 축 — 팀 코치 보너스와 합산.
- **밸런스 완화 장치**: 보너스 폭이 소속팀 [②자원](리그팀/05_팀_특성_시스템.md)에 **반비례** — 궁핍한 팀 소속일수록 개인 투자의 효용이 크고, 이미 부유한 팀 소속이면 한계효용이 작다. "부익부 빈익빈"이 아니라 **"돈으로 팀 격차를 스스로 좁힌다"**는 방향 — 예를 들어 투수코치가 없는 학교라면 "기술 코치"를 사비로 고용해 정확히 그 공백을 메우는 선택이 가능해진다.
- 해제하면 즉시 비용 중단, 보너스도 즉시 소멸(누적 저장 없음 — 계속 유지해야 의미 있는 소비).

### 3-3. 투자 (시즌말 자동 알림, 스킵 가능)

- **시즌 종료 시점 자동 알림**(선택지, 스킵 가능) — "저축(안전, 낮은 확정 수익) / 투자(공격적, 원금 손실 가능하나 기대 수익 높음) / 그대로 둠(스킵, 기본값)" 3택. 응답 안 해도 자동으로 "저축"과 동일하게 처리되어 진행을 막지 않음([콘텐츠/02_이벤트](콘텐츠/02_이벤트.md) §6 비긴급 메시지 패턴과 동일).
- 매 시즌 가벼운 확률 롤 1회 → 은퇴 시 최종 자산(§4)에 누적 반영.
- **투자 수익은 과세, 손실은 공제**(§2 세율 재사용) — "번 돈은 다 과세 대상"이라는 일관성. 손실 시에는 세금 없이 그대로 차감(이미 손해 봤는데 세금까지 물리진 않음).
- 정확한 수익률·손실 확률은 스탯 스케일 확정 후.
