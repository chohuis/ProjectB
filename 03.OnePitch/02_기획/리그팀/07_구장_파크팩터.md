# 구장 · 파크팩터

> 근거: [세계 개요](00_개요.md) · [육성코어/07_매치_엔진](../육성코어/07_매치_엔진.md) §10-1(파크팩터가 인플레이 결과 확률에 가중) · [03_대학](03_대학.md) §4-1(거점구장 방식 선례)
> 상태: **구조 확정(초안)** · 정확한 확률 보정 수치는 스탯 스케일 확정 후
> 목적: [07_매치_엔진](../육성코어/07_매치_엔진.md) §10-1이 "구단에 구장 데이터가 있어야 파크팩터가 존재한다"고 전제한 것을 채운다. [01_프로](01_프로.md) §9 열린세부(팀별 홈구장) 해소.

## 1. 배치 원칙 — 리그마다 다름

| 리그 | 방식 | 구장 수 |
|---|---|---|
| 프로 | 팀별 전용구장 | 10 |
| 대학 | 거점구장 공유(기존 5조 구조 재사용) | 5 |
| 고교 | 8권역 거점구장 공유(신규) | 8 |
| 독립 | 거점구장 공유(신규, 4지역+상무 전용) | 5 |

- 172팀(고교102·대학50·독립10) 전부가 아니라 **28개 구장**만 정의하면 전 리그가 커버된다 — 대학이 이미 증명한 "거점구장 공유" 패턴을 고교·독립까지 확장.

## 2. 파크팩터 3유형

| 유형 | 성격 |
|---|---|
| **타자친화** | 좁은 구장 — 홈런 확률↑ |
| **투수친화** | 넓은 구장 — 홈런 확률↓, 수비범위 중요도↑ |
| **중립** | 표준 |

- [07_매치_엔진](../육성코어/07_매치_엔진.md) §6 인플레이 결과 확률에 가중치로 반영. 정확한 보정폭은 스탯 스케일 확정 후.

## 3. 프로 (10구장, 팀별 전용)

| 팀 | 구장 | 파크팩터 |
|---|---|---|
| 서울 로열스 | 로열파크 | 중립 |
| 서울 가디언스 | 가디언스필드 | 투수친화 |
| 서울 코브라스 | 코브라돔 | 타자친화 |
| 인천 샤크스 | 샤크스파크 | 중립 |
| 수원 나이츠 | 나이츠아레나 | 타자친화 |
| 대전 팬텀스 | 팬텀스타디움 | 중립 |
| 광주 팬서스 | 팬서스파크 | 타자친화 |
| 대구 세이버스 | 세이버스필드 | 투수친화 |
| 부산 웨이브스 | 웨이브스타디움 | 타자친화 |
| 창원 스타스 | 스타스파크 | 투수친화 |

- **서울 가디언스(투수왕국) = 투수친화구장**: 팀 철학과 구장 성격이 일치하는 예 — 우연이 아니라 자연스러운 조합.

## 4. 대학 (5거점구장 — 기존 명칭에 파크팩터만 추가)

[03_대학](03_대학.md) §4-1에 이미 있는 5조 거점구장 그대로, 파크팩터만 신규 배정.

| 거점구장 | 소속 조 | 파크팩터 |
|---|---|---|
| 미르구장(서울) | A조(서울6+경인4) | 중립 |
| 별빛구장(경인) | B조(경인10) | 타자친화 |
| 금강구장(충청) | C조(충청8+강원2) | 투수친화 |
| 노을구장(호남) | D조(호남7+대경3) | 타자친화 |
| 태종구장(영남) | E조(부경울7+대경2+제주1) | 중립 |

## 6. 독립 (4지역 거점구장 + 상무 전용, 신규)

시민구단 9개는 지역거점 4곳에 배분(§1 대학 패턴 재사용), 상무 피닉스는 무연고라 전용 구장.

| 거점구장 | 소속 팀 | 파크팩터 |
|---|---|---|
| 강변구장(수도권) | 서울 레이븐스·서울 코메츠·파주 유니콘스·인천 그리폰스 | 타자친화 |
| 계백구장(중부: 충청+강원) | 대전 블레이즈·원주 위너스 | 중립 |
| 남녘구장(남부: 호남+대구경북+부경울) | 광주 라이노스·대구 호크스·부산 씨라이온스 | 투수친화 |
| 창공구장(상무 전용) | 상무 피닉스 | 중립 |
