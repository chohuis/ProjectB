# 팀 특성 시스템 (3슬롯 조합)

> 근거: [세계 개요](00_개요.md) · 대화 설계(2026-07-09)
> 상태: **구조 확정(초안) — 육성코어 9/9 완료로 연결 시스템 전부 존재. 정확한 수치만 스탯 스케일 확정 후**
> 목적: 135개 배경팀(고교90·대학40·독립5) 하나하나에 개별 시나리오를 쓰지 않고도, **3특성 조합**으로 절차적 정체성을 부여해 "전 팀 선택 가능"을 실현한다. content.db의 "규칙 기반 런타임 생성" 원칙을 선수뿐 아니라 **팀 정체성**에도 적용한 것.

## 1. 3 카테고리 (고정 슬롯 — 각 1개씩 조합)

### ① 운영 철학 (12개) — 성장 시스템에 영향

| 특성 | 성격 |
|---|---|
| 육성중심 | 유망주 개발 우선 |
| 공격야구(화력) | 타격·득점력 중심 |
| 수비/짜임새 | 실점 최소화, 조직력 |
| 스몰볼 | 번트·도루 등 상황 야구 |
| 근성/언더독 | 정신력·투지 |
| 데이터중심 | 분석·현대적 접근 |
| 투수왕국 | 투수 육성 특화 |
| 전통/정통 | 보수적 기본기 중시 |
| 스파르타(혹독훈련) | 성장 빠르나 부상 위험↑ |
| 부상방지/재활특화 | 부상 확률↓·회복↑ |
| 베테랑우대 | 경험 신뢰, 신인 기회 적음 |
| 젊은피(세대교체) | 신인 기회 많음 |

### ② 자원(재정) (4개) — 계약·시설 시스템에 영향

| 특성 | 성격 |
|---|---|
| 부유 | 시설·계약금·FA 영입력 최상 |
| 안정 | 평균 |
| 알뜰 | 궁핍하지만 효율적으로 운영(페널티 적음) |
| 궁핍 | 재정 열악 |

### ③ 위상(등급) (7개) — 주목도·압박(진행동력) 시스템에 영향

| 특성 | 성격 |
|---|---|
| 슈퍼스타 | 최상급 명문, 최고 압박 |
| 명문 | 전통 강호 |
| 엘리트 | 상위권, 안정적 |
| 신흥 | 상승세, 다크호스 |
| 중견 | 중간 |
| 언더독 | 약체, 반란 서사 |
| 몰락한명문 | 한때 최강이었으나 추락 — 재건 서사 전용 |

**조합 수**: 12×4×7 = **336가지** (배경팀 135개 커버에 충분, 겹침도 자연스러운 현상으로 허용).

## 2. 효과 매핑 — 슬롯마다 다른 시스템

| 슬롯 | 건드리는 시스템 | 효과 방향(수치는 스탯 스케일 확정 후) |
|---|---|---|
| **① 철학** | 성장(육성) | 능력치군별 성장 가중(예: 투수왕국=구위·구종↑), 부상·회복 보정(부상방지↓ 부상, 스파르타=trade-off), 신인/베테랑 기회 배분 |
| **② 자원** | 계약·시설 | 계약금·연봉 규모(고교 지명·FA), 시설 투자(성장·회복 보너스), 로스터 뎁스(백업 질) |
| **③ 위상** | 주목도·압박 | 시작 주목도 보정, **팬 기대치 압박 강도**([개요 §4](../00_게임_개요.md) 진행동력 시스템의 다이얼), 미디어 노출 빈도 |

## 3. 이벤트 연동 — 특성이 콘텐츠 생성 규칙이 됨

- content.db 이벤트는 `trigger` 조건으로 발동 — **팀 특성을 트리거 조건에 포함**.
- **특성 전용 이벤트**: 특성 하나당 전용 이벤트 몇 개 (예: 스파르타→"혹독한 특타 지시", 몰락한명문→"왕년의 영광을 아느냐").
- **조합 전용 이벤트**(선택): 특이 조합에만 뜨는 이벤트 (예: 궁핍×몰락한명문→자조 이벤트).
- 172팀이 서로 다른 이벤트 폭을 갖게 되어 **반복 방지**.

## 4. 배정 방식

### 4-1. 선택 가능 팀 + 프로(27+10=37팀) — 수동 배정

이미 손으로 쓴 정체성 텍스트가 있는 팀(고교12·대학10·독립 육성형4+상무1·프로10)은 **절차적으로 굴리지 않고, 기존 텍스트에 맞춰 특성 3개를 수동 배정**.

예: 백호고(투수 육성 명문) → [투수왕국 × 안정 × 엘리트] / 서울 로열스(최강·자금력) → [전통/정통 × 부유 × 슈퍼스타] / 한라고(제주 유일·최소자원) → [근성/언더독 × 궁핍 × 언더독]

### 4-2. 배경팀(135개) — 전력★ 가중 랜덤

이미 부여된 전력★(1~5)과 **②자원·③위상을 가중 연동**, **①철학은 전력★ 무관 자유 배치**.

| 전력★ | 위상 쏠림 | 자원 쏠림 |
|---|---|---|
| ★★★★★ | 슈퍼스타·명문 위주 | 부유 위주 |
| ★★★★ | 명문·엘리트 위주 | 부유·안정 위주 |
| ★★★ | 엘리트·신흥·중견 고르게 | 안정 위주 |
| ★★ | 중견·언더독 위주 | 안정·알뜰 위주 |
| ★ | 언더독·몰락한명문 위주 | 알뜰·궁핍 위주 |

- "가중"이지 "고정"이 아님 — 가끔 ★★★★★인데 "언더독" 위상이 나오는 **이변 조합**도 허용(재미 포인트).
- 철학은 전력★와 무관 — 약팀도 "투수왕국" 철학일 수 있음(다만 결과가 안 좋을 뿐).

## 5. 적용 대상 요약

| 리그 | 선택(수동) | 배경(절차) |
|---|---:|---:|
| 고교 | 12 | 90 |
| 대학 | 10 | 40 |
| 독립 | 5(육성형4+상무1) | 5(도전형) |
| 프로 | 10 | — |

- **독립 10팀 구성** = 시민구단 9(육성형4 + 도전형5) + 상무 1. 수동배정 대상 = 육성형4(⭐)+상무1, 배경(절차) = 도전형5.
- 배경(절차) 합계 = 90+40+5 = **135개** — §목적문·§4-2와 일치. 전체 팀(2군 제외) 172개 = 선택·수동 37 + 배경 135.
