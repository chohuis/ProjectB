- **독립 10팀**(20→10 축소): 4단계 생존 리그(라운드마다 하위권 탈락) 구조 채택 — 상세는 [리그팀/04_독립](04_독립.md).

- **프로 2군**: 1군 연동(같은 연고). **상무**: 독립리그 10번째 팀으로 배치(군 소재, 중립) — [04_독립](04_독립.md) 참조.

## 3. 선택 가능 팀 (주인공 시작·진학)

- **고교 선택 = 12교**: 각 권역 대표 강호 1교(8) + 야구 강세 권역 추가 4(수도권2·영남1·호남1).
- **대학 선택 = 10~12교**: 유사 분포.

## 4. 문서 관리 (리그별 통합)

각 리그 = **팀 + 리그·대회 운영을 한 파일**로 통합. 공통(권역·분포·시뮬)만 이 개요에.
```
02_기획/리그팀/
├── 00_개요.md   ← 프레임워크(8권역·분포·시뮬 범위)  [공통]
├── 01_프로.md   ← 프로 팀(1군10+2군10) + 운영
├── 02_고교.md   ← 고교 팀(102) + 리그·대회 운영
├── 03_대학.md   ← 대학 팀(50) + 운영
└── 04_독립.md   ← 독립 팀(10) + 운영
```

## 4-1. 시뮬 범위 (모든 리그 공통) — 하이브리드

171팀을 전부 실제 경기시키지 않는다. 주인공 거리에 따라 해상도 차등:

| 반경 | 대상 | 처리 |
|---|---|---|
| **풀** | 주인공 소속 리그/권역 + 참가 대회 | 실제 경기 시뮬 |
| **간이** | 인접 리그(상위 무대 등) | 전력 기반 결과·순위만 |
| **결과만** | 먼 리그/권역 | 순위표 산출(라이벌·연감용) |

- 기술 설계의 반경 시뮬과 연동. 주인공 스테이지 이동 시 활성 범위도 이동.
- **예외 — 고교**: 규모 한정(102팀·3년)이라 **전부 풀 시뮬**. 하이브리드는 대규모 스테이지(프로+팜)용.

## 5. 팀 명명 정책 (결정 필요)

- 가상 세계(실명 아님) → 팀명·연고 **가공**.
- 핵심 팀(프로 10, 선택 고교·대학)은 **지금 상세 명명**, 배경 팀(나머지 ~150)은 **카운트 + 생성 규칙**으로 두고 콘텐츠 단계에서 채움.
