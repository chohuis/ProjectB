## 3. 정규시즌 (마라톤, 144경기)

- **단일 리그**(디비전 없음) — 10팀이 서로 **16차전(홈8·원정8)** = 팀당 **144경기**, 리그 전체 720경기.
- 고교·대학·독립보다 훨씬 긴 호흡 — 주인공이 가장 오래 머무는 무대인 만큼 가장 묵직한 마라톤 감각 채택.
- 순위로 5강(포스트시즌 진출권) 결정.

## 4. 포스트시즌 — 5강 와일드카드 사다리

| 라운드 | 대진 | 전제 |
|---|---|---|
| 와일드카드전 | 4위 vs 5위 | 단판(**5위는 2승, 4위는 1승만 해도 진출** — 정규시즌 상위 어드밴티지) |
| 준플레이오프 | 3위 vs WC승자 | 5전3승제 |
| 플레이오프 | 2위 vs 준PO승자 | 5전3승제 |
| **한국시리즈**(챔피언결정전) | **1위 직행** vs PO승자 | 7전4승제 |

- **정규시즌 1위 특혜**: 포스트시즌 없이 최종 결승 직행 — 144경기 마라톤을 지배한 팀에게 주는 가장 큰 보상.
- 독립리그 포스트시즌(준PO→PO→챔피언, 사다리 원형)의 **완전판** — 프로가 원조, 독립은 축소 버전.

## 5. 퓨처스(2군) 리그

- **단일 리그**(10팀 전부, 북남 분리 없음).
- **경기 수**: 각 상대 11차전 = 9팀×11 = **99경기**(약 100경기). 1군(144)보다 가벼움 — 2군 본래 역할(육성·대타 평가)에 집중.
- **간이 포스트시즌**: 상위 4팀, **단판 사다리**(3위vs4위 → 승자vs2위 → 승자vs1위). 1군 5강처럼 무겁게 안 감 — 2군은 나름의 우승 서사만 갖추는 수준.

## 6. 경기 규칙

- **투구수**: 아마추어(대학·독립)와 동일 **120구 상한 + 동일 휴식표** 유지 — 전 리그 일관성.
- 콜드게임 없음(프로는 9이닝 완주가 원칙 — 아마추어 규정과 차별화).

## 7. 시즌 캘린더

11월 통합 드래프트로 들어온 신인이 다음 스프링캠프부터 바로 합류하도록 배치.

| 시기 | 진행 |
|---|---|
| 11월 | **통합 드래프트**(고교·대학·독립 신인 영입) → 시즌 종료 직후 |
| 12~2월 | **스토브리그**(FA·트레이드·연봉협상) + 신인 합류 |
| 2월 말 | 스프링캠프·시범경기 |
| **3월** | **정규시즌 개막**(144경기) |
| 7월 중순 | 올스타전(브레이크) |
| 7월 말 | **트레이드 데드라인**(연 1회) |
| 10월 말 | 정규시즌 종료 → 5강 확정 |
| 10월 말~11월 초 | **포스트시즌**(WC→준PO→PO→한국시리즈) |
| 11월 | 시즌 종료 → 곧바로 통합 드래프트 |

- 1군·2군 시즌은 **동기화**(같은 3월 개막~10월 종료 창 안에서 진행, 2군은 경기 수만 적어 먼저 마무리).

## 8. 시뮬 범위 (프로 = 전부 풀)

- 고교·대학·독립과 동일 원칙 — **1군720경기 + 2군990경기, 전부 경기 단위 풀 시뮬**. 규모는 가장 크지만 세계 전체(고교102·대학50·독립10·프로1군10+2군10)의 시뮬 일관성을 위해 하이브리드 예외 없이 통일. → [00_개요 §4-1](00_개요.md).
- (엔진 성능 최적화는 구현 단계 과제로 이월 — 설계 차단 아님.)
