# 프로 커리어 (Phase 2~5)

> 근거: [커리어 구조](01_커리어_구조.md) · 백지 재정의 Q&A (2026-07-09)
> 상태: **Phase 2~5 확정(초안) — 시장·계약/히스토리는 별도 시스템**

## Phase 2 — 입단 & 2군

- **입단**: 지명 → 계약(계약금·연봉, 지명 라운드 반영) → **2군 배치**. 특급 신인은 **1군 직행 여지**(조건부).
- **2군 리그(10팀)**: 육성·실전. 선발/불펜 역할 경험, 성적·성장 누적 → **콜업 자격** 축적.

## Phase 3 — 1군 정착 & 역할

### 승강 (1군↔2군) — 상황 기반 (FM 현실)
- 2군에서 잘해도 **1군 자리(부상·부진·일정)가 열려야 기회**. 판정 = 성적 + 팀 상황 + 운.
- 1군 부진 → 강등. **상시 경쟁** — "닥닥함과 기회의 맛".
- **체크 시점 = 이벤트 트리거형(상시)**: 고정 주기가 아니라 **1군 자리 공백 발생 시점**(부상자 발생·로스터 조정)마다 콜업 후보 판정. "부진 누적"은 [09_평가_시스템](육성코어/09_평가_시스템.md) **월 단위 평가**("이달의 페이스") 나쁨이 강등 후보 편입의 신호로 연결.
- **콜업 후보 선정(구체)**: 공백이 생긴 포지션(선발/불펜 등)과 같은 역할의 2군 투수 중 [09_평가_시스템](육성코어/09_평가_시스템.md) 월단위 평가 상위자를 우선 후보로 삼는다. 복수 후보가 겹치면 **감독 신뢰도(아래) → 최근 페이스 → 시즌 누적 성적** 순으로 우선순위.

### 역할 획득 — 감독 배정 (신뢰형)
- 역할(선발/불펜(추격조→셋업)/마무리/스윙맨)은 **감독이 배정**.
- 플레이어는 **성적·구위로 신뢰를 쌓아** 원하는 역할로 이동. 수동적·현실적.
- **NPC 투수도 같은 로테이션 자리를 두고 경쟁** — NPC는 S~D 등급 없이 능력치·누적성적으로 감독 AI가 서열화([09_평가_시스템](육성코어/09_평가_시스템.md) §5-2). 주인공만 등급이라는 추가 렌즈로 감독 신뢰를 쌓는다.
- **감독 신뢰도 = 관계도 재사용**(대화 설계 2026-07-14): 새 수치를 만들지 않고 [콘텐츠/01_캐릭터](콘텐츠/01_캐릭터.md) §5 관계도(-100~100, 7단계 라벨)를 감독↔주인공 관계에 그대로 적용한다 — 좋은 성적이 관계도를 올리고, 감독 의견에 반하는 선택([07_매치_엔진](육성코어/07_매치_엔진.md) §8 투수 교체 강행 등)은 내린다. 역할 배정·콜업 우선순위는 이 관계도 라벨이 높을수록 유리.
- **역할 경쟁에서 주인공의 강점**: NPC는 순수 능력치·누적성적 서열뿐이지만, 주인공은 여기에 관계도(감독과의 개인적 신뢰)라는 추가 렌즈가 더해진다 — [09_평가_시스템](육성코어/09_평가_시스템.md) §5-2 "S~D 등급은 오직 주인공만의 서사적 피드백 렌즈"와 같은 원리.

### 정착
- 꾸준한 1군 기여 → 주전화 → Phase 4.

## Phase 4 — 주전·전성기 & 갈림길 B (진행 중)

- 에이스화, 타이틀·올스타·명성.
- **갈림길 B**: FA / 트레이드 / 잔류·재계약 (**해외 제외 — DLC 이월**).
- ※ FA·계약·협상 상세는 [시장·계약](06_시장_계약.md)에서 확정 완료(방출·연봉재계약·FA·트레이드·계약조항).

## Phase 5 — 베테랑·노쇠·재기 (확정: 가변 강도)

- **노쇠 = 가변** — 관리·유형·부상 이력에 따라 하락 속도 다름:
  - 구위 의존형(파이어볼러) → 상대적으로 빨리 저묾.
  - 기교파(제구·변화구) → 오래 감, **유종의 미** 가능.
  - 관리(휴식·부상 예방)·역할 조정이 수명을 좌우 → 전략적 후반부.
- 구위 하락 → 역할 하향(선발→불펜→추격조), 기교파 전환.
- 부상·방출·강등 위기, **재기 서사** → 은퇴(Phase 6).
