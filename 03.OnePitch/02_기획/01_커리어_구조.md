# 커리어 구조 · 코어 루프

> 근거: [게임 개요](00_게임_개요.md) · 백지 재정의 Q&A (2026-07-09)
> 상태: **구조 확정 · 페이즈별 상세 진행 중**

## 1. 코어 루프 (확정)

- **시간 = 일 단위 continue** (FM식). 하루씩 진행하다 개입 지점에서 자동 정지(엔진 PendingAction).
- **훈련·접근법은 지속 설정**. **범위 = 풀 라이프**(아마 → 은퇴), **총 15~20시즌**.

```
[지속 설정] 훈련·접근법 → Continue(하루씩) → [개입] 경기·선택·이벤트·부상 → 결정 → 반복
→ 시즌 → 커리어 아크 → 은퇴 → 인생 기록
```

## 2. 커리어 아크 (확정 골격)

```
Phase 1 아마추어 (고교 시작, 17세)
   └ 갈림길 A: 프로 지명 / 대학(4년) / 독립 / 입대(병역)
Phase 2 프로 육성 (2군)
Phase 3 1군 정착 & 역할 축
Phase 4 주전·전성기 (국내 정상)
   └ 갈림길 B: FA / 트레이드 / 잔류·재계약   ※ 해외 진출은 이번 제외 → 추후 DLC
Phase 5 베테랑·노쇠
Phase 6 은퇴 → 인생 기록
```

- **커리어 정점 = 국내 1군**(이번 스코프). 해외 상위 리그는 **DLC로 이월**.

## 4. 아마 대회 구조

- **지역 주말리그** (권역별 정규 리그, 전·후반기) — 꾸준한 정규 경기.
- **전국 토너먼트 대회 여러 개** — 일부 전 팀 개방형, 일부 리그 상위 진출제. 전국 무대 임팩트 = 주목도 급상승.
- **3년 누적**(성적 + 대회 임팩트) → **주목도(스카우트 시선)** → 드래프트 가치.

## 5. 갈림길 상세

| 갈림길 | 시점 | 경로 |
|---|---|---|
| **A** | 고교 졸업 | 프로 지명 / 대학(4년) / 독립 / **입대(병역)** |
| **B** | 전성기 | FA / 트레이드 / 잔류·재계약 (해외 제외) |
| (상시) | 프로 내내 | 승강(1군↔2군), 부상 후 재활 선택, 계약 갱신 |

- **병역**: 한국 남성 의무. 입대(현역) 또는 커리어 중 **상무(선수 복무)** 등으로 소화 — 커리어 관통 요소, 상세는 Phase별로.

## 6. 페이즈 기간 (대략, 성장 가변)

아마 3년(+대학4/독립 시 연장) / 2군 1~4년 / 1군 정착 2~3년 / 전성기 5~8년 / 노쇠 2~4년 → 은퇴. **총 15~20시즌**.

## 7. 관통 3요소

| 요소 | 작동 |
|---|---|
| **역할 축** | 선발/불펜/마무리/스윙맨 — 성적·구위·신뢰로 변동 (Phase 3~5) |
| **갈림길** | A(진로)·B(전성기) + 상시(승강·재활·계약·병역) |
| **승강** | 1군↔2군 상시 (부진→강등, 회복→콜업) |
