프로 전 단계. **고교에서 시작** → 갈림길 A에서 프로 지명 / **대학**(연장 4년) / **독립**(재도전) / **입대**([병역](03_병역.md)) 분기.

## A. 고교 (Phase 1 시작)

### A-1. 시작
- **고교 1학년, 17세, 신입생.** 능력 낮음·잠재력 히든.
- 선택 학교(강호 **12교**)의 지역·전력·스타일·기존 에이스가 시작 조건·서사를 가른다.

### A-2. 리그 세계
- **전국 8권역, 102팀**(선택 12 + 배경 90, 전부 풀 시뮬 — 저해상도 아님). 상세: [리그팀/02_고교](리그팀/02_고교.md).

### A-3. 시즌 캘린더 (매 학년)
| 시기 | 내용 |
|---|---|
| 봄 | 지역 주말리그 전반기 + **개나리기** |
| 여름 | **장미기** + **무궁화기** |
| 가을 | 주말리그 후반기 + **국화기** + **패왕기** |
| 겨울 | 비시즌(훈련·성장) / 3학년 말: 드래프트 |

### A-4. 전국 토너먼트 대회 (5개, 꽃 이름)
| 대회 | 꽃/시기 | 성격 |
|---|---|---|
| **개나리기** | 개나리·봄 | 시즌 첫 전국 무대(개막) |
| **장미기** | 장미·초여름 | 정통 선수권 |
| **무궁화기** | 무궁화·한여름 | 대형 대회(나라꽃) |
| **국화기** | 국화·가을 | 전 팀 개방형(언더독 기회·최다 경기) |
| **패왕기** | 늦가을 | 주말리그 권역 상위만 진출, 그 해 최강전 |

- **주말리그**(권역별 정규, 전·후반기)가 정규 경기 뼈대. 운영 상세 → [리그팀/02_고교](리그팀/02_고교.md) §4.

### A-5. 3년 성장
- 1→2→3학년. 후보→주전→에이스. 실력·주목도·팀 역할 성장 → 갈림길 A.

---

## B. 대학 (아마 연장, 4년) — 고교급 상세 무대

- **진학 시 대학 선택**: 선택 가능 10개 대학(전국 50팀 — [리그팀/03_대학](리그팀/03_대학.md)).
- **대학 리그**(권역 정규) + **전국 대학 대회** — 고교에 준하는 상세 무대.
- **4년 진행**(1~4학년) → **대졸 드래프트**로 재평가.
- 장점: 완성도·몸 만들 시간, 즉시전력 기대 / 단점: 4년 소요, 정체·부상 리스크, 프로 간 동기보다 늦음.
- 병역: 대학 중·후 처리 가능.

---

## C. 독립 (재도전 무대) — 전국 10팀

- **전국 10팀** 독립 리그, **4단계 생존 리그**(라운드마다 하위권 탈락 → 최종 4팀 포스트시즌). 미지명자·늦깎이·재도전. 상세: [리그팀/04_독립](리그팀/04_독립.md).
- 리그전 중심. 성적 내면 → **프로 지명(신인·중도)** or 실업/방출.
- 장점: 실전 지속·재도전 / 단점: 무대 수준·주목도 낮음, 불확실. 보통 1~2년 내 승부.

---

## D. 드래프트 · 주목도 (공통)

- **주목도** = 리그 성적(방어율·탈삼진·이닝 — **경기별 [09_평가_시스템](육성코어/09_평가_시스템.md) S~D 등급 누적**) + 대회 임팩트(호투·우승) + 마일스톤. **무대 수준 가중**(고교 < 대학, 독립 낮음).
- **드래프트**: 3학년/대학4년/독립 시점에 1군 **10팀 라운드별 지명**. 주목도 → 라운드·픽 순위(상위 = 계약금·기대↑).
- 지명 → 프로(Phase 2, 2군 시작 / 특급 1군 직행 여지). **미지명 → 대학·독립·입대 재경로**.

## E. 갈림길 A 종합

| 경로 | 소요 | 특징 |
|---|---|---|
| 프로 지명 | 즉시 | 최단 경로, 지명 가치가 시작 조건 |
| 대학 | 4년 | 완성도↑·재평가, 늦음 |
| 독립 | 1~2년 | 재도전, 불확실 |
| 입대 | ~2년 | 병역 조기 이행(주로 현역) → [병역](03_병역.md) |
