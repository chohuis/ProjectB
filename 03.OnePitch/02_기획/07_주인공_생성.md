# 주인공 생성 (뉴게임)

> 근거: [게임 개요](00_게임_개요.md) §5(1인칭 원칙) · [커리어 구조](01_커리어_구조.md) Phase 1 시작점 · [01_선수_능력치](육성코어/01_선수_능력치.md) · [05_구종_시스템](육성코어/05_구종_시스템.md) §3 · [리그팀/00_개요](리그팀/00_개요.md) §3 · [리그팀/02_고교](리그팀/02_고교.md) · 대화 설계(2026-07-14)
> 상태: **구조 확정(초안)** · 정확한 수치는 스탯 스케일 확정 후
> 목적: Phase 1(고교 1학년, 17세) 진입 전, 플레이어가 자신의 투수를 만드는 최초 화면 흐름을 정의한다. 모든 육성 시스템(능력치·구종·성장)의 **초기값을 채우는 유일한 관문**.

## 0. 생성 철학 — 하이브리드

- **핵심 몇 가지만 직접 선택**(이름·좌우투·지역·학교·투수 타입), **나머지는 배경에서 타입에 연동된 가중 랜덤**(능력치 초기값). 히든스탯만 완전 비공개 랜덤.
- [00_게임_개요](00_게임_개요.md) §3 "FM형 관리·육성"·"화면은 거칠게" 원칙을 생성 단계에도 적용 — 손맛은 주되, 능력치를 포인트 배분하듯 직접 조립하게 하지 않는다.
- [01_선수_능력치](육성코어/01_선수_능력치.md) §4 히든스탯 원칙(UI 비노출, 행동으로 간접 노출)이 생성 시점에도 깨지지 않는다.

## 1. 생성 플로우 (순서)

```
1. 이름 입력
2. 좌완/우완 선택
3. 지역 선택 (8권역 중 1곳)
4. 학교 선택 (선택 권역 내 고교 전부, 전력★·팀 특성 등 정보 전부 공개)
5. 투수 타입 선택 (4종 중 1개) → 시작 능력치 배경 배정
6. 시작 구종 확인 — 포심 패스트볼 고정 + 두 번째 구종 후보(타입 테마 × 학교 코치 가중) 중 선택(스킵 가능)
7. 요약 화면 (이름·좌우투·학교·타입·구종 확인) → 게임 시작
```

- 히든스탯은 화면에 등장하지 않고 전 과정이 끝난 시점에 배경에서 완전 별개로 배정된다(§5).
- **지역 → 학교** 2단계인 이유: [리그팀/00_개요](리그팀/00_개요.md) §2 8권역 분포(고교 102팀, 권역당 6~20팀)가 이미 있는 구조라, 102개를 한 리스트로 나열하지 않고 권역으로 먼저 좁혀서 탐색 부담을 줄인다.
- 학교(4단계)가 투수 타입(5단계)보다 앞에 오는 이유: 시작 구종 후보(§6)가 학교(코치 구성)와 타입 둘 다에 연동되므로, 학교가 먼저 정해져야 이후 계산이 가능하다.

## 2. 좌완/우완 — 소폭 시스템 영향

- 실제 야구의 "좌완 프리미엄"을 반영 — 좌완은 희귀성으로 **주목도·시장가치에 소폭 가산**([09_평가_시스템](육성코어/09_평가_시스템.md) 주목도, [06_시장_계약](06_시장_계약.md) FA·트레이드 평가에 소폭 보정치로 연동). 구종 습득·마스터리에는 영향 없음(카탈로그·습득 조건 동일).
- 정확한 보정폭은 스탯 스케일 확정 후.

## 3. 지역 → 학교 선택 — 전국 102교 전부 선택 가능

[리그팀/00_개요](리그팀/00_개요.md) §3이 확정한 대로 **102교 전부가 시작 학교 후보**다. 상위 기획에서 학교를 "특별 취급 vs 일반 취급"으로 나누지 않는다 — 강한 학교, 중간 학교, 약한 학교가 전력★(1~5)과 [팀 특성 3슬롯](리그팀/05_팀_특성_시스템.md)(철학·자원·위상)만으로 다양하게 존재할 뿐, 선택 화면·게임 동작에서 학교군 사이에 별도 분기를 두지 않는다.

### 3-1. 1단계 — 지역 선택

[리그팀/00_개요](리그팀/00_개요.md) §1의 8권역(서울/경기·인천/강원/충청/호남/대구·경북/부산·경남·울산/제주) 중 1곳을 먼저 고른다. 권역별 총 팀 수가 화면에 요약 표시된다.

### 3-2. 2단계 — 학교 선택 (권역 내, 전부 공개)

선택한 권역 소속 고교 전체가 하나의 리스트로 뜬다(예: 서울 16교, 제주 6교 — [리그팀/02_고교](리그팀/02_고교.md) §1 권역별 분포). 모든 학교에 대해 전력★·[팀 특성 3슬롯](리그팀/05_팀_특성_시스템.md)(철학·자원·위상)을 동일한 형식으로 전부 공개한 채 선택 — "이 학교는 최강이지만 에이스 경쟁이 치열하다, 저 학교는 최약체지만 언더독 서사다"처럼 트레이드오프를 알고 고르는 전략적 선택.

- 일부 학교는 [리그팀/02_고교](리그팀/02_고교.md) §2에 별도로 정리된 서사 훅 텍스트가 이미 준비돼 있어 화면에 함께 노출되지만, 이는 콘텐츠 분량 차이일 뿐이다 — 어떤 학교를 고르든 선택 가능 여부·이후 게임 동작(코치 배치·훈련 효율·시장 반응 등)은 동일한 시스템(§ 아래)으로 처리된다.
- 학교 선택이 실제로 좌우하는 것(전부 기존 시스템 재사용, 신규 메커니즘 없음): [02_스태프_능력치](육성코어/02_스태프_능력치.md) 코치 배치(성장 효율·구종 후보), [05_팀_특성_시스템](리그팀/05_팀_특성_시스템.md) 3슬롯(훈련 강도 기본값·시설 보너스·시작 주목도).
- **학교는 능력치 초기값 자체에는 영향을 주지 않는다**(§4는 투수 타입에만 연동) — 시작부터 학교 간 스탯 격차가 벌어지지 않고, 격차는 이후 성장 속도·코치 품질에서 벌어진다.

## 4. 투수 타입 (아키타입) — 4종, 시작 능력치와 구종 후보를 동시에 결정

9개 노출 능력치([01_선수_능력치](육성코어/01_선수_능력치.md) §2) 중 **피지컬 2개(구속·체력)·기술 1개(제구)·멘탈 1개(침착함)**를 각각 대표하는 4타입.

| 타입 | 우세 스탯 | 구종 후보(§6) |
|---|---|---|
| **강속구형** | 구속(+구위 소폭) | 패스트볼류만(투심·커터) |
| **제구형** | 제구(+경기운영 소폭) | 오프스피드류·브레이킹볼류 전 계열 개방 — 유일하게 패스트볼 밖 구종에 접근 |
| **체력형** | 체력·회복력 | 패스트볼류만(투심·커터) |
| **돌부처형** | 클러치·침착함 | 패스트볼류만(투심·커터) |

- **시작 능력치 배정**: 9개 스탯 전부 20~80 스케일 중 낮은 구간에서 시작하되, 타입의 우세 스탯 1~2개는 **상단 밴드**(예상 28~35대), 나머지는 **하단 밴드**(예상 20~28대)에서 랜덤 — "무명에서 시작"은 유지하면서 타입별 개성이 드러남. 정확한 밴드 경계는 스탯 스케일 확정 후.
- 학교·좌우투와 무관하게 타입만이 능력치 분포를 가른다 — 어느 학교를 고르든 4타입은 동일하게 선택 가능.
- 히든스탯(§5)에는 영향 없음 — 타입은 노출 능력치·구종에만 연동.
- **왜 제구형만 예외인가**: 제구는 "커맨드로 타이밍을 뺏는" 기술이라 다양한 구종을 초반부터 구사하는 것과 서사적으로 맞물린다. 나머지 세 타입(구속·체력·멘탈 중심)은 아직 기교보다 힘·꾸준함·멘탈로 승부하는 신인이라는 설정 — 패스트볼류 밖 구종은 이후 [훈련 시스템](육성코어/06_훈련_시스템.md)으로 습득.

## 5. 히든스탯 — 완전 비공개 랜덤

- 천재성·인성·성실함([01_선수_능력치](육성코어/01_선수_능력치.md) §4) 전부 **생성 화면에 등장하지 않고 배경에서 랜덤 배정**. 학교·좌우투·타입·구종 선택과 무관.
- 플레이어는 자기 자신의 히든스탯도 완전히 모른 채 시작 — [01_선수_능력치](육성코어/01_선수_능력치.md) §8 열린세부 "히든 스탯이 주인공 본인에게도 완전 비공개인지"를 **완전 비공개로 확정**. 코치 피드백·이벤트·성장 패턴 관찰로만 짐작 가능.

## 6. 시작 구종 — 포심 고정 + 타입별 제한된 두 번째 구종 (기본 1개, 최대 2개)

[05_구종_시스템](육성코어/05_구종_시스템.md) §3 "신인은 1~2개만 갖고 시작(예: 포심 단독, 또는 포심+체인지업류)"을 구체화 — 새 메커니즘 아님. 시작 구종은 **많아야 2개**로 확정.

- **1구종(고정)**: 포심 패스트볼 — 전 주인공 공통, 선택지 없음.
- **2구종(선택, 스킵 가능)**: 타입에 따라 후보 풀 자체가 다르다 — "포심만으로 시작"도 항상 선택 가능.

| 타입 | 후보 풀 | 후보 개수 |
|---|---|---|
| 강속구형·체력형·돌부처형 | **패스트볼류만**(투심·커터 2종) | 2개 그대로 제시 |
| 제구형 | 오프스피드류·브레이킹볼류(체인지업·포크볼·싱커·슬라이더·커브·스위퍼 6종) | 학교 코치 가중 랜덤으로 **3~4개** 제시 |

- 학교 코치 보너스([02_스태프_능력치](육성코어/02_스태프_능력치.md) "가르칠 수 있는 구종")는 후보 노출 확률·순서에 반영 — 제구형이 해당 구종 전문 코치가 있는 학교를 고르면 그 구종이 후보에 더 자주 뜬다. 강속구형·체력형·돌부처형은 후보 풀이 투심·커터 2개뿐이라 학교 보너스의 체감 폭이 작음.
- 너클볼은 항상 후보에서 제외 — [05_구종_시스템](육성코어/05_구종_시스템.md) §1 "이질적 메커니즘이라 단독 분류"된 특수구이므로 시작 후보 밖, 이후 훈련으로만 습득 가능.
- 선택한 두 번째 구종은 마스터리 1단계("습작")에서 시작.
