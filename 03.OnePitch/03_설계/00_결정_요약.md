# 설계 결정 요약 (Decision Registry)

> 환경·데이터·구조에서 **확정된 모든 결정의 단일 목록**. 상세는 각 문서 링크.
> 일자: 2026-07-09 · 상태: 환경·데이터·구조 설계 완료 (다음 = 기획)

## A. 게임 정체성 (방향 확정 — 상세는 기획 단계)

- 장르: 투수 인생 시뮬레이션(1인 육성), 샌드박스.
- 철학: 경기 = S~D 평가 / **인생 = 기록(히스토리) = 엔딩** / 정답 루트 없음.
- 스코프: 동일 장르 정식화 (프로토 계승 + 정식 품질).

## B. 환경 / 플랫폼 → [01_환경](01_환경.md)

- 타깃: **PC(Win/Mac/Linux) + Android**, web 제외.
- UI: **Flutter(Dart)** / 엔진: **Rust 네이티브 lib** / 경계: **flutter_rust_bridge(FFI)**.
- 프로토(Svelte UI·Rust 로직)는 **모티브로만**, 재작성.
- 저장: 앱 내부저장소(`path_provider`) + **로컬 자동 롤링 백업만**(시즌시작 3 + 마이그레이션 전).
- 콘텐츠 번들: content.db 앱번들, Android는 첫 실행·버전변경 시 추출.
- 업데이트: 스토어(PC 인스톨러 / Android Play), `content_version`으로 OTA 미래 가능(서버 미구축).
- 빌드: **단일 Dart CLI**(`dart run tool gen/run/build/content/test/release`).
- 매치 렌더: Flame 또는 CustomPainter (UI 기획서에서 확정).

## C. 데이터 → [02_데이터](02_데이터.md)

- **데이터 3곳, 전부 엔진 뒤**: `content.db`(정의) / `slot.db`(상태) / **Rust 상수**(은닉 밸런스).
- content.db: **SQLite 직접 저작**(원본=DB), 평칼럼+JSON컬럼, FK 무결성, 런타임 메모리 로드.
- AI 대량생성: JSON → `import`(가산·dry-run), `dump`로 git 이력.
- slot.db: 파일=슬롯, 5그룹, **능력치 stats·xp JSON 동거**, rusqlite.
- 세이브: **처음부터 단계별 마이그레이션**(meta.save_version).
- 무결성: **HMAC 서명+감지**(핵심테이블 해시→meta, 키 Rust 내장, 위반 시 '수정됨' 표시·차단 안 함).
- **D4-2**: 개방(콘텐츠·서사) vs 은닉(매치확률·성장계수·평가경계 = Rust 상수).

## D. 구조 / 아키텍처 → [03_구조](03_구조.md)

- **계층 계약**: UI = 렌더+입력만(로직 0·데이터 직접접근 0) / Rust = **유일 권위**(계산·난수·상태변이·대사해결) / frb 타입드 경계.
- 엔진 API: **범용 쿼리 + 커맨드**(화면별 뷰 아님). 매치 = frb 불투명 핸들 세션. 진행정지 = `PendingAction` 목록.
- Flutter: **Riverpod**(얇은 캐시) + **go_router** + feature-first + **반응형**(처음부터).
- 폴더: `app/`(Flutter) + `engine/`(Rust crate).

## E. 다국어 → [02_데이터 §다국어](02_데이터.md)

- **초반부터**, 런칭 ko + en.
- 크롬 = Flutter i18n(Dart) / 콘텐츠 = content.db **번역키 테이블**(strings, 업계 표준).
- 엔진이 `setLocale`로 로케일 해결해 반환. 이름풀 = 로케일 데이터(번역 아님). 폴백 ko.

## F. 남은 것 (설계 차단 아님)

- **P7 frb 빌드 스파이크** — 구현 착수 직전 실측(빌드·rusqlite·content.db 추출).
- 구현 세부: content.db 컬럼 확정 · Repository 커맨드 시그니처 · AI import 규약.
- **뒤로 미룸**: 변조방지 수준 · 모바일 UX 상세(DotPitchStage 터치) · 배포채널·수익화 · ICU MessageFormat.
