[Academic Session System]
작성일: 2026-03-05
목표: 고등학교~대학교 학업 루프를 정의하고, 학습/튜터링/과제 선택이 능력치와 기록에 반영되도록 설계한다.

---------------------------------------------
1. 데이터 구조
---------------------------------------------
1) AcademicSessionSlot
   - SlotId: morning / support / evening (3개 고정)
   - Label: 수업 / 보충·튜터링 / 과제·자습
   - AvailableActions: EducationAction 목록(학년/시기/상태에 따라 필터)

2) EducationAction
   - Id / Name / Description
   - Type: lecture / tutoring / assignment / rest
   - Requirements: 집중도, 피로, 경고 상태 등의 조건
   - Effects: Knowledge / Focus / Fatigue / ScholarshipPoint / WarningPoint 변화
   - Visibility: 특정 구간(시험 주간, 성적 위험 등)에서만 노출

3) AcademicDayPlan
   - Date / SlotAssignments[3]
   - AutoRecommend: true/false (자동 추천 여부)

4) AcademicState
   - KnowledgeScore (0~100)
   - FocusScore (0~100)
   - 내신 등급 (0~4.5) : 중간/기말/과제/출석 기반으로 계산
   - ScholarshipPoint / WarningPoint
   - RecentActions Log

---------------------------------------------
2. 일일 진행 흐름
---------------------------------------------
1) 오전(수업)
   - 선택지: 수업을 듣는다 / 수업을 대충 듣는다 / 잔다
   - 효과 예시:
     * 듣는다: Knowledge +0.3~0.6, Focus +0.1, Fatigue +1
     * 대충: Knowledge +0.1~0.3, Focus -0.1, Fatigue +0.5
     * 잔다: Knowledge 0, Focus +0.2, WarningPoint +1

2) 오후(보충·튜터링)
   - 선택지: 추가 학습을 한다 / 영어 회화를 한다 / 잔다
   - 효과 예시:
     * 추가 학습: Knowledge +0.4~0.7, Focus -0.1, Fatigue +1.5
     * 영어 회화: Knowledge +0.2~0.4, Focus +0.1, Fatigue +1
     * 잔다: Knowledge 0, Focus +0.2, WarningPoint +1

3) 저녁(과제·자습)
   - 선택지: 과제를 한다 / 과제를 대충한다 / 잔다
   - 효과 예시:
     * 과제: Knowledge +0.2~0.4, Focus -0.1, Fatigue +1
     * 대충: Knowledge +0.1, Focus -0.2, WarningPoint +0.5
     * 잔다: Knowledge 0, Focus +0.2, WarningPoint +1

---------------------------------------------
3. 성적 계산
---------------------------------------------
- KnowledgeScore와 FocusScore는 일일 선택에 따라 누적/감소한다.
- 시험 성적은 기본 Knowledge + Focus 보정으로 산출한다.
- 내신 등급은 중간/기말/과제/출석의 가중 평균으로 계산한다.
- ScholarshipPoint는 일정 기준 이상에서 획득, WarningPoint는 누적 시 경고 이벤트 트리거.

---------------------------------------------
4. 추천/자동 배치
---------------------------------------------
- AutoRecommend가 켜진 경우, 현재 피로/성적/경고 상태를 기반으로 추천 선택지를 배치한다.
- 시험 주간에는 공부/과제 쪽 가중치를 상향한다.

---------------------------------------------
5. UI/로그 반영
---------------------------------------------
- 선택 결과는 하루 로그와 메시지 카드로 기록한다.
- 내신 등급 변동/경고/장학 이벤트는 메시지 인박스에 적재한다.

---------------------------------------------
6. 밸런싱 가이드
---------------------------------------------
- 고교 기준: Knowledge 성장 속도는 느리게, Focus 소모는 상대적으로 크게.
- 시험 주간: 성적 영향 1.2~1.4배 보정.
- 경고 누적: 3회 경고 시 패널티 이벤트 발생.
