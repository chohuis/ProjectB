[KBO Draft System - Game Plan]

1. 후보 풀 구성
- 고교 3학년, 재수생, 일부 대학/독립 출신 포함
- 각 후보는 시즌 기록, PreferredRole, 스카우트 코멘트 보유

2. 드래프트 방식
- 10개 프로 구단이 라운드별 순번제로 지명 (총 10라운드, 100명)
- 순번은 전년도 성적 역순 또는 단일 리그 기준

3. 구단 스카우트 성향
- `PKT_*.txt` 데이터 참고: 팀별 선호 포지션/스타일/팜 상황 반영
- 스카우트 점수 계산 → 각 라운드에서 우선순위 결정

4. 드래프트 진행
- 라운드별 루프: 남은 후보 중 팀 성향 점수가 높은 선수 지명
- 주인공이 지명되면 해당 팀 이벤트 발생, 미지명 시 대체 경로 선택

5. 대체 경로
- 대학 진학: 누적 학점 기준 충족 시 지원 가능
- 독립리그: 독립 팀 트라이아웃으로 이동
- 군입대: 화천 7사단 입대 (18개월 Phase 시스템)

6. 리그/대회 연계
- 시즌 중 등장한 라이벌/팀메이트 NPC가 드래프트에서도 등장
- 경기 로그/스카우트 기록을 드래프트 후보 평가에 활용
